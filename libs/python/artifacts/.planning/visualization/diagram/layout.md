# [PY_ARTIFACTS_DIAGRAM_LAYOUT]

`DiagramLayout` folds one admitted adjacency frame into the positioned `DiagramGlyph` sequence for every `DiagramKind`. Admission accumulates independent column, identity, endpoint, parent, constraint, and geometry faults before graph construction; valid zero remains distinct from absent evidence. One stable `rustworkx.PyDiGraph` node index keys coordinates, routes, and marks. Layout owns coordinates and edge routes; `DiagramDraw` owns target emission, and the data graph owner retains analysis.

`LayoutPolicy` closes force, radial, layered, projected, and constrained placement under one total dispatch. `ForceMode`, `RingMode`, `HierarchyEngine`, `ProjectionKind`, and `LayoutRule` carry each growth axis as data. `ELK` owns orthogonal sections, ports, and nested containers; selecting the glyphset `EdgeRoute.ORTHOGONAL` regime resolves the engine to `HierarchyEngine.ELK`, and the ELK result's per-port coordinates read back into each emitted `Port.at`, so a rendered seat and a routed endpoint share one engine truth. Area-bearing kinds preserve measurement evidence: stacking bands aggregate admitted program area, program boxes scale by `sqrt(area)`, and site rings retain shoelace magnitude. One synchronous `_render` kernel crosses the runtime lane under the `KernelTrait` its resolved policy derives — GIL-releasing native engines ride the thread arm, GIL-holding pure-Python engines the warm process pool, sub-quantum projected and constrained transforms the loop — and returns positioned glyph substrate without minting receipt identity.

## [01]-[INDEX]

- [01]-[LAYOUT]: `DiagramLayout`'s coordinate-assignment fold over a `data/graph#GRAPH` adjacency frame — one typed column admission per `DiagramKind`, coordinates via the `LayoutPolicy` the kind selects (force, radial, four-engine layered, projected AEC transforms, plan-anchored constrained), the near-peer node-link kinds emitted through one `_standard` generator over `_EMIT` policy rows and the V15 area-law kinds through their own arms, composed solar furniture on the `SUN_PATH` arm, offloaded to the runtime thread lane.

## [02]-[LAYOUT]

- Owner: `DiagramLayout` owns one `LayoutPolicy` dispatch over `DiagramKind` and one `LayoutFault` admission rail. Typed node evidence carries plan coordinates, program area, site polygons, nested-layout policy, edge cardinality, and solar-site atmosphere; `LayoutMap`, `RouteMap`, and `SeatMap` retain the stable graph index. `assign` re-spells the accumulated fault family onto `BoundaryFault.config`.
- Admission: `_admitted` accumulates every independent `LayoutFault` before graph construction. Required node columns, optional numeric columns when present, edge weights, unique node identity, endpoint resolution, parent resolution, constraint references, stacking row roles, site polygons, and all-or-none solar-site evidence cross this boundary once. `_num` reads admission-proven values; `_opt` preserves genuine absence. `weight` absence selects the declared style width, and `shell` absence selects one radial ring.
- Cases: `Force`/`Radial`/`Layered`/`Projected`/`Constrained`, one total `match` over the policy `tag`; `Force.mode` selects the `_FORCE` placement row, `Layered.engine` the `_ENGINE` provider row, `Radial.mode` the `_RING` row, `Projected.kind` the `PROJECTION` transform; `Constrained` seeds plan-anchored (typed `east`/`north` under a `strong` stay, every other node the `circular_layout` spread under a `weak` stay) then folds the `LayoutRule` set to `kiwisolver` `Constraint`s; `CIRCULATION` and `SECTION_CALLOUT` select `Constrained()` in `_KIND_POLICY` because their marks are plan-anchored building geometry, the `Force()` default that scattered rooms by topology being the rejected form. Solve-quality evidence (`Constraint.violated()` tallies, edit-variable re-solve) is `composition/sheet#SHEET`'s interactive axis — this owner runs one batch solve and stops.
- Emit: the near-peer kinds share one generator — `_standard` folds nodes and edges once over the `_EMIT` `EmitStyle` policy row (layer names, label keys, `NodeSizing` fixed/area-true/entity dims, edge width, `EdgeInk` fixed-versus-source tint, default caps), per-edge `weight` and `source_cardinality`/`target_cardinality` columns overriding by value — so `NODE_LINK`/`FLOWCHART`/`ENTITY_RELATION`/`SANKEY`/`PROGRAM`/`CIRCULATION` are six rows, not six comprehension bodies, and a new near-peer kind is one `_EMIT` row plus one `_KIND_POLICY` row with zero new emission code. Structurally distinct kinds keep their own arms: `SUN_PATH` (solar marks + furniture + compass), `STACKING` (floor bands + area segments), `SITE` (true polygons + callouts), `SECTION_CALLOUT` (detail frames + references).
- Auto: `assign` offloads the synchronous `_render` kernel through `LanePolicy.offload(Kernel.of(_render, _trait(policy)))` and preserves the runtime rail — `_ENGINE_TRAIT` answers isolation per layered engine, so the pure-Python grandalf/pyelk passes cross `HOSTILE` onto the warm process pool instead of stalling the loop disguised as a `RELEASING` thread kernel, while sub-quantum projected and constrained transforms ride `INLINE`. `_position` resolves orthogonal layered policy to `HierarchyEngine.ELK` even when callers construct the union case directly. `_sugiyama_layout` carries crossing minimization, isolated nodes, and dummy-chain routes; `_elk_layout` carries nested containers, fixed ports, absolute routed sections, per-ordinal route keys (a port-bound endpoint id never parses as a node index), and the port-seat harvest `_seated` folds onto the emitted marks. `SUN_PATH` composes `project` and `furniture`; solar furniture requires the complete `latitude`/`longitude`/`year`/`timezone` site group, while angle-only frames emit marks without fabricated site geometry.
- Growth: a new diagram kind is one `DiagramKind` row plus one `_EMIT` row (or one distinct `_emit` arm when structurally new) plus one `_KIND_POLICY` selection and one `_REQUIRED` contract row, never a new layout method family; a new layered engine one `HierarchyEngine` member plus one `_ENGINE` row, never a fifth `LayeredPolicy` case; a new force placement one `ForceMode`/`_FORCE` pair; a new AEC projection one `ProjectionKind` member plus a `_solar_arc`-shaped transform plus a `PROJECTION` row; a new ring mode one `RingMode`/`_RING` pair; a new constraint one `LayoutRule` case plus one `_rule_constraints` arm; a new container engine one `_SUB_ALGORITHM` row; a new cardinality token one `_CARDINALITY` row; a new route style rides the engine that owns it.
- Boundary: no SVG emission (`visualization/diagram/draw#DRAW`'s), no graph analysis (`data/graph#GRAPH`'s centrality/shortest-path/community kernel stays at the data plane; this owner runs only layout functions), no ephemeris computation (`visualization/diagram/solar#SOLAR` solves sun positions; this owner composes `project`/`furniture`), no ad-hoc color (the `GlyphStyle` indices key the `graphic/color/derive#DERIVE` palette), no receipt and no content key (draw's); `rustworkx` owns force/radial/topological placement, `grandalf`/`fast-sugiyama` Sugiyama, `pyelk` ports/nesting/orthogonal, `kiwisolver` the Cassowary solve, all behind one `LayoutPolicy` owner — a hand-rolled Sugiyama/spring/constraint loop, an `ORTHOGONAL` aliased to a straight-line router, a synchronous native layout left on the event loop, a GIL-holding pure-Python engine pass declared `RELEASING`, a `Force()` default on a plan-anchored kind, a fixed rectangle standing for a measured area, and an absent measurement coerced to `0.0` are the rejected forms.

@tagged_union(frozen=True)
class LayoutRule:
    # constraint vocabulary the Constrained policy folds onto the kiwisolver Solver; each case names the node indices and strength band.
    tag: Literal["align", "separate", "distribute", "anchor", "mirror"] = tag()
    align: tuple[tuple[int, ...], RuleAxis, RuleStrength] = case()
    separate: tuple[int, int, RuleAxis, float, RuleStrength] = case()
    distribute: tuple[tuple[int, ...], RuleAxis, float, RuleStrength] = case()
    anchor: tuple[int, Point, RuleStrength] = case()
    mirror: tuple[int, int, RuleAxis, float, RuleStrength] = case()

    @staticmethod
    def Align(nodes: tuple[int, ...], axis: RuleAxis = "x", strength: RuleStrength = "strong") -> "LayoutRule":
        return LayoutRule(align=(nodes, axis, strength))

    @staticmethod
    def Separate(before: int, after: int, axis: RuleAxis = "x", gap: float = 48.0, strength: RuleStrength = "required") -> "LayoutRule":
        return LayoutRule(separate=(before, after, axis, gap, strength))

    @staticmethod
    def Distribute(nodes: tuple[int, ...], axis: RuleAxis = "x", gap: float = 64.0, strength: RuleStrength = "medium") -> "LayoutRule":
        return LayoutRule(distribute=(nodes, axis, gap, strength))

    @staticmethod
    def Anchor(node: int, at: Point, strength: RuleStrength = "required") -> "LayoutRule":
        return LayoutRule(anchor=(node, at, strength))

    @staticmethod
    def Mirror(left: int, right: int, about: float, axis: RuleAxis = "x", strength: RuleStrength = "medium") -> "LayoutRule":
        return LayoutRule(mirror=(left, right, axis, about, strength))

    def nodes(self) -> tuple[int, ...]:
        match self:
            case LayoutRule(tag="align", align=(nodes, _, _)) | LayoutRule(tag="distribute", distribute=(nodes, _, _, _)):
                return nodes
            case LayoutRule(tag="separate", separate=(before, after, _, _, _)):
                return (before, after)
            case LayoutRule(tag="anchor", anchor=(node, _, _)):
                return (node,)
            case LayoutRule(tag="mirror", mirror=(left, right, _, _, _)):
                return (left, right)
            case _ as unreachable:
                assert_never(unreachable)

    def valid(self) -> bool:
        match self:
            case LayoutRule(tag="align", align=(nodes, _, _)):
                return len(nodes) > 1
            case LayoutRule(tag="separate", separate=(before, after, _, gap, _)):
                return before != after and isfinite(gap) and gap >= 0.0
            case LayoutRule(tag="distribute", distribute=(nodes, _, gap, _)):
                return len(nodes) > 1 and len(nodes) == len(frozenset(nodes)) and isfinite(gap) and gap >= 0.0
            case LayoutRule(tag="anchor", anchor=(_, (x, y), _)):
                return isfinite(x) and isfinite(y)
            case LayoutRule(tag="mirror", mirror=(left, right, _, about, _)):
                return left != right and isfinite(about)
            case _ as unreachable:
                assert_never(unreachable)


@tagged_union(frozen=True)
class LayoutPolicy:
    tag: Literal["force", "radial", "layered", "projected", "constrained"] = tag()
    force: tuple[int, int, ForceMode] = case()
    radial: tuple[float, RingMode] = case()
    layered: LayeredPolicy = case()
    projected: tuple[ProjectionKind, float] = case()
    constrained: tuple[LayoutRule, ...] = case()

    @staticmethod
    def Force(seed: int = 0, iterations: int = 100, mode: ForceMode = ForceMode.SPRING) -> "LayoutPolicy":
        return LayoutPolicy(force=(seed, iterations, mode))

    @staticmethod
    def Radial(scale: float = 1.0, mode: RingMode = RingMode.CIRCULAR) -> "LayoutPolicy":
        return LayoutPolicy(radial=(scale, mode))

    @staticmethod
    def Layered(
        direction: LayerDirection = "TB",
        engine: HierarchyEngine = HierarchyEngine.FAST_SUGIYAMA,
        route: EdgeRoute = EdgeRoute.LINES,
        crossing: Crossing = "median",
    ) -> "LayoutPolicy":
        # ORTHOGONAL is ELK's native capability; force the engine so it never degrades to a straight-line alias on a Sugiyama engine.
        resolved = HierarchyEngine.ELK if route is EdgeRoute.ORTHOGONAL else engine
        return LayoutPolicy(layered=LayeredPolicy(direction=direction, engine=resolved, route=route, crossing=crossing))

    @staticmethod
    def Projected(kind: ProjectionKind, scale: float = 1.0) -> "LayoutPolicy":
        return LayoutPolicy(projected=(kind, scale))

    @staticmethod
    def Constrained(*rules: LayoutRule) -> "LayoutPolicy":
        return LayoutPolicy(constrained=rules)

    def valid(self) -> bool:
        match self:
            case LayoutPolicy(tag="force", force=(_, iterations, _)):
                return iterations > 0
            case LayoutPolicy(tag="radial", radial=(scale, _)) | LayoutPolicy(tag="projected", projected=(_, scale)):
                return isfinite(scale) and scale > 0.0
            case LayoutPolicy(tag="layered"):
                return True
            case LayoutPolicy(tag="constrained", constrained=rules):
                return all(rule.valid() for rule in rules)
            case _ as unreachable:
                assert_never(unreachable)

class DiagramLayout(Struct, frozen=True):
    kind: DiagramKind
    adjacency: pl.DataFrame
    attributes: pl.DataFrame
    # `lane` arrives projected via LanePolicy.of(context) at the composition root — a capacity literal has no owner.
    lane: LanePolicy
    policy: LayoutPolicy | None = None

    async def assign(self) -> RuntimeRail[tuple[DiagramGlyph, ...]]:
        # coordinate substrate for draw; layout mints no receipt and no key.
        crossed = await self.lane.offload(Kernel.of(self._render, self._trait(self.policy or _KIND_POLICY[self.kind])))
        return crossed.bind(lambda inner: inner.map_error(lambda fault: BoundaryFault(config=(f"diagram.layout.{self.kind}", fault.tag))))

    def _trait(self, policy: LayoutPolicy, /) -> KernelTrait:
        # isolation truth per resolved policy: PURE is barred outright — this module's polars/rustworkx/kiwisolver
        # natives never survive an isolated subinterpreter import — so a GIL-holding engine crosses HOSTILE instead.
        match policy:
            case LayoutPolicy(tag="layered", layered=layered):
                return _ENGINE_TRAIT[HierarchyEngine.ELK if layered.route is EdgeRoute.ORTHOGONAL else layered.engine]
            case LayoutPolicy(tag="projected") if self.kind is DiagramKind.SUN_PATH:
                return KernelTrait.RELEASING  # pvlib/numpy solar furniture is a GIL-releasing numpy-band pass
            case LayoutPolicy(tag="projected") | LayoutPolicy(tag="constrained"):
                # per-node float transforms and one batch Cassowary solve are sub-quantum at diagram scale
                return KernelTrait.INLINE
            case _:
                return KernelTrait.RELEASING  # rustworkx force/radial placement is native Rust

    def _render(self) -> Result[tuple[DiagramGlyph, ...], LayoutFault]:
        policy = self.policy or _KIND_POLICY[self.kind]
        return self._admitted(policy).bind(lambda graph: self._emit(graph, policy, *self._position(graph, policy)))

    def _admitted(self, policy: LayoutPolicy, /) -> Result[rx.PyDiGraph, LayoutFault]:
        projected = (
            ()
            if policy.tag == "projected"
            and policy.projected[0] is ProjectionKind.FLOOR_BAND
            and self.kind is DiagramKind.STACKING
            else _PROJECTED_REQUIRED[policy.projected[0]]
            if policy.tag == "projected"
            else ()
        )
        site_columns = ("latitude", "longitude", "year", "timezone")
        site_axes = (*site_columns, "site_altitude", "pressure", "temperature")
        site = site_columns if any(column in self.attributes.schema for column in site_axes) else ()
        required = tuple(dict.fromkeys(("id", *_REQUIRED[self.kind], *projected, *site)))
        schema, edge_schema = self.attributes.schema, self.adjacency.schema
        missing = tuple(LayoutFault(missing_column=column) for column in required if column not in schema)
        nulls = tuple(LayoutFault(null_column=column) for column in required if column in schema and self.attributes[column].null_count() > 0)
        numeric = tuple(LayoutFault(non_numeric=column) for column in _NUMERIC & set(schema) if not schema[column].is_numeric())
        finite = tuple(
            LayoutFault(invalid_value=f"{column}:{index}:{value}")
            for column in _NUMERIC & set(schema)
            if schema[column].is_numeric()
            for index, value in enumerate(self.attributes[column].to_list())
            if value is not None and (isinstance(value, bool) or not isfinite(float(value)))
        )
        measures = tuple(
            LayoutFault(invalid_value=f"{column}:{index}:{value}")
            for column, frame in (("area", self.attributes), ("weight", self.adjacency))
            if column in frame.schema and frame.schema[column].is_numeric()
            for index, value in enumerate(frame[column].to_list())
            if value is not None and (isinstance(value, bool) or not isfinite(float(value)) or float(value) < 0.0)
        )
        site_values = tuple(
            LayoutFault(invalid_value=f"site:{row.get('id')}:{field}:{value}")
            for row in self.attributes.to_dicts()
            if site
            for field, value in (("year", row.get("year")), ("timezone", row.get("timezone")))
            if (field == "year" and (not isinstance(value, int | float) or isinstance(value, bool) or not isfinite(float(value)) or not float(value).is_integer()))
            or (field == "timezone" and (not isinstance(value, str) or not value))
        )
        ids = tuple(self.attributes["id"].to_list()) if "id" in schema else ()
        invalid_ids = tuple(LayoutFault(invalid_value=f"id:{value!r}") for value in ids if not _keyable(value))
        identity = tuple(LayoutFault(duplicate_id=str(value)) for index, value in enumerate(ids) if value in ids[:index])
        id_set = frozenset(value for value in ids if _keyable(value))
        edge_columns = ("source", "target") if len(self.adjacency) else ()
        edge_missing = tuple(LayoutFault(missing_column=column) for column in edge_columns if column not in edge_schema)
        edge_nulls = tuple(LayoutFault(null_column=column) for column in edge_columns if column in edge_schema and self.adjacency[column].null_count() > 0)
        # every identity-bearing read past here admits through `_keyable` FIRST: an unhashable raw value — a
        # list-valued id, endpoint, or parent — lands as its own fault row, never a TypeError inside a
        # frozenset membership or dict build.
        endpoints = tuple(
            LayoutFault(unknown_endpoint=str(value)) if _keyable(value) else LayoutFault(invalid_value=f"{column}:{value!r}")
            for column in edge_columns
            if column in edge_schema
            for value in self.adjacency[column].to_list()
            if value is not None and (not _keyable(value) or value not in id_set)
        )
        edge_numeric = (
            (LayoutFault(non_numeric="weight"),)
            if "weight" in edge_schema and not edge_schema["weight"].is_numeric()
            else ()
        )
        parents = tuple(
            LayoutFault(unknown_parent=str(parent)) if _keyable(parent) else LayoutFault(invalid_value=f"parent:{parent!r}")
            for row in self.attributes.to_dicts()
            if (parent := row.get("parent")) is not None and (not _keyable(parent) or parent not in id_set)
        )
        parent_map = {row["id"]: row["parent"] for row in self.attributes.to_dicts() if _keyable(row.get("id")) and _keyable(row.get("parent"))}
        parent_cycles = tuple(LayoutFault(cycle=f"parent:{start}") for start in parent_map if start in _parent_chain(parent_map, start))
        port_owner = {row["id"]: frozenset(port.id for port in _ports_of(row)) for row in self.attributes.to_dicts() if _keyable(row.get("id"))}
        edge_ports = tuple(
            LayoutFault(invalid_value=f"port:{owner}:{name}")
            for row in self.adjacency.to_dicts()
            for endpoint, ref in (("source", "source_port"), ("target", "target_port"))
            # a non-keyable owner already carries its endpoints fault; its port reference is unresolvable noise, skipped
            if (name := row.get(ref)) is not None and _keyable(owner := row.get(endpoint)) and name not in port_owner.get(owner, frozenset())
        )
        rules = policy.constrained if policy.tag == "constrained" else ()
        invalid_rules = tuple(
            LayoutFault(invalid_rule=str(node)) for rule in rules for node in rule.nodes() if node not in range(len(ids))
        )
        invalid_policy = () if policy.valid() else (LayoutFault(invalid_value=f"policy:{policy.tag}"),)
        stacking = tuple(
            LayoutFault(invalid_value=f"stacking:{row.get('id')}")
            for row in self.attributes.to_dicts()
            if self.kind is DiagramKind.STACKING
            and not (
                (isinstance(row.get("level"), int | float) and not isinstance(row.get("level"), bool) and row.get("parent") is None)
                or (
                    isinstance(row.get("area"), int | float)
                    and not isinstance(row.get("area"), bool)
                    and _keyable(row.get("parent"))
                    and row.get("parent") in id_set
                )
            )
        )
        # a SITE row with NEITHER polygon field has no geometry to lay out — the per-field shape scan below only
        # judges present values, so absence faults here before graph construction.
        polygonless = tuple(
            LayoutFault(invalid_value=f"ring|footprint:{row.get('id')}")
            for row in self.attributes.to_dicts()
            if self.kind is DiagramKind.SITE and row.get("ring") is None and row.get("footprint") is None
        )
        polygons = tuple(
            LayoutFault(invalid_value=f"{key}:{row.get('id')}")
            for row in self.attributes.to_dicts()
            if self.kind is DiagramKind.SITE
            for key in ("ring", "footprint")
            if (raw := row.get(key)) is not None
            and (
                not isinstance(raw, list | tuple)
                or len(raw) < 3
                or not all(
                    isinstance(point, list | tuple)
                    and len(point) == 2
                    and all(
                        isinstance(value, int | float) and not isinstance(value, bool) and isfinite(float(value))
                        for value in point
                    )
                    for point in raw
                )
            )
        )
        faults = (*missing, *nulls, *numeric, *finite, *measures, *site_values, *invalid_ids, *identity, *edge_missing, *edge_nulls, *endpoints, *edge_numeric, *parents, *parent_cycles, *edge_ports, *invalid_rules, *invalid_policy, *stacking, *polygonless, *polygons)
        if faults:
            return Error(faults[0] if len(faults) == 1 else LayoutFault(many=faults))
        graph = self._as_graph()
        return (
            Error(LayoutFault(cycle="rustworkx"))
            if policy.tag == "layered"
            and policy.layered.engine is HierarchyEngine.RUSTWORKX
            and policy.layered.route is not EdgeRoute.ORTHOGONAL
            and not rx.is_directed_acyclic_graph(graph)
            else Ok(graph)
        )

    def _as_graph(self) -> rx.PyDiGraph:
        graph: rx.PyDiGraph = rx.PyDiGraph()
        rows = self.attributes.to_dicts()
        indices = graph.add_nodes_from(rows)
        lookup = dict(zip((row["id"] for row in rows), indices, strict=True))
        graph.add_edges_from((lookup[r["source"]], lookup[r["target"]], r) for r in self.adjacency.to_dicts())
        return graph

    def _position(self, graph: rx.PyDiGraph, policy: LayoutPolicy) -> LayoutResult:
        match policy:
            case LayoutPolicy(tag="force", force=(seed, iterations, mode)):
                pos = _FORCE[mode](graph, seed, iterations)
                return {i: _pt(pos[i]) for i in graph.node_indices()}, {}, {}
            case LayoutPolicy(tag="radial", radial=(scale, mode)):
                pos = _RING[mode](graph, scale)
                return {i: _pt(pos[i]) for i in graph.node_indices()}, {}, {}
            case LayoutPolicy(tag="layered", layered=layered):
                engine = HierarchyEngine.ELK if layered.route is EdgeRoute.ORTHOGONAL else layered.engine
                return _ENGINE[engine](graph, layered)
            case LayoutPolicy(tag="projected", projected=(kind, scale)):
                transform = PROJECTION[kind]
                indices = (
                    tuple(i for i in graph.node_indices() if _opt(graph.get_node_data(i), "level").is_some())
                    if kind is ProjectionKind.FLOOR_BAND
                    else graph.node_indices()
                )
                return {i: transform(graph.get_node_data(i), scale) for i in indices}, {}, {}
            case LayoutPolicy(tag="constrained", constrained=rules):
                return _constrained_layout(graph, rules)
            case _ as unreachable:
                assert_never(unreachable)

    def _emit(
        self, graph: rx.PyDiGraph, policy: LayoutPolicy, coords: LayoutMap, routes: RouteMap, seats: SeatMap
    ) -> Result[tuple[DiagramGlyph, ...], LayoutFault]:
        match self.kind:
            case DiagramKind.SUN_PATH:
                scale = _proj_scale(policy)
                ticks = tuple(
                    DiagramGlyph(marker=MarkerMark(x=coords[i][0], y=coords[i][1], kind=MarkerKind.TICK, angle=_num(graph.get_node_data(i), "azimuth"), style=GlyphStyle("sun")))
                    for i in graph.node_indices()
                )
                arcs = tuple(
                    DiagramGlyph(edge=EdgeMark(source=s, target=t, points=_edge_points(routes, coords, s, t), label=None, style=GlyphStyle("path", stroke=1)))
                    for s, t in graph.edge_list()
                )
                north = DiagramGlyph(marker=MarkerMark(x=0.0, y=0.0, kind=MarkerKind.NORTH, angle=0.0, style=GlyphStyle("compass")))
                return _sun_furniture(graph, scale).map(lambda furnishing: (*furnishing, *arcs, *ticks, north))
            case DiagramKind.STACKING:
                _roots, kids = _nest(graph)
                floors = tuple(i for i in graph.node_indices() if _opt(graph.get_node_data(i), "level").is_some())
                bands = tuple(
                    DiagramGlyph(swimlane=SwimlaneMark(
                        index=i,
                        x=0.0,
                        y=coords[i][1],
                        w=max(_NODE_W, _floor_total(graph, kids.get(i, ())) * _AREA_UNIT),
                        h=_BAND_H,
                        title=_label(graph.get_node_data(i), "floor"),
                        style=GlyphStyle("floors", fill=i),
                    ))
                    for i in floors
                )
                segments = tuple(segment for i in floors for segment in _segments(graph, i, kids.get(i, ()), coords[i][1]))
                return Ok((*bands, *segments))
            case DiagramKind.SITE:
                scale = _proj_scale(policy)
                polygons = tuple(
                    mark
                    for i in graph.node_indices()
                    for d in (graph.get_node_data(i),)
                    for mark in (
                        _polygon(d, "ring", scale, i, _label(d, "label"), "parcels"),
                        _polygon(d, "footprint", scale, i, None, "buildings"),
                    )
                    if mark is not None
                )
                callouts = tuple(
                    DiagramGlyph(annotation=AnnotationMark(x=coords[i][0], y=coords[i][1], text=text, anchor=TextAnchor.MIDDLE, style=GlyphStyle("callouts")))
                    for i in graph.node_indices()
                    if (text := _label(graph.get_node_data(i), "label")) is not None
                )
                return Ok((*polygons, *callouts))
            case DiagramKind.SECTION_CALLOUT:
                frames = tuple(
                    DiagramGlyph(swimlane=SwimlaneMark(index=i, x=coords[i][0], y=coords[i][1], w=160.0, h=120.0, title=_label(graph.get_node_data(i), "detail"), style=GlyphStyle("frame", fill=i)))
                    for i in graph.node_indices()
                )
                refs = tuple(
                    DiagramGlyph(annotation=AnnotationMark(x=coords[i][0], y=coords[i][1], text=text, anchor=TextAnchor.MIDDLE, style=GlyphStyle("callouts")))
                    for i in graph.node_indices()
                    if (text := _label(graph.get_node_data(i), "reference")) is not None
                )
                return Ok((*frames, *refs))
            case kind:
                return Ok(_standard(graph, coords, routes, seats, _EMIT[kind], _route_of(policy)))


# --- [OPERATIONS] -----------------------------------------------------------------------
def _route_of(policy: LayoutPolicy, /) -> EdgeRoute:
    return policy.layered.route if policy.tag == "layered" else EdgeRoute.LINES


def _parent_chain(parents: dict[object, object], start: object, /) -> frozenset[object]:
    # ancestors reachable from `start`; membership of `start` itself proves a parent cycle at admission time.
    seen: set[object] = set()
    node = parents.get(start)
    while node is not None and node not in seen:
        seen.add(node)
        node = parents.get(node)
    return frozenset(seen)


def _standard(
    graph: rx.PyDiGraph, coords: LayoutMap, routes: RouteMap, seats: SeatMap, style: EmitStyle, route: EdgeRoute = EdgeRoute.LINES, /
) -> tuple[DiagramGlyph, ...]:
    # ONE near-peer emission generator: six kinds are _EMIT rows over this body, never six comprehension copies.
    # admission-proven parent relations and named port seats SURVIVE emission — draw's container parenting,
    # cycle rejection, and drawio seat lookup consume the same topology ELK nested on, never a flattened copy.
    _, kids = _nest(graph)
    owner = frozendict({child: parent for parent, children in kids.items() for child in children})
    nodes = tuple(
        DiagramGlyph(node=NodeMark(
            index=i,
            x=coords[i][0],
            y=coords[i][1],
            w=w,
            h=h,
            label=_label(d, style.label_key),
            style=GlyphStyle(style.node_layer, fill=i),
            shape=style.shape or _shape_of(d),
            ports=_seated(_ports_of(d), i, seats),
            parent=owner.get(i),
        ))
        for i in graph.node_indices()
        for d in (graph.get_node_data(i),)
        for w, h in (_sized(style, d),)
    )
    edges = tuple(
        DiagramGlyph(edge=EdgeMark(
            source=s,
            target=t,
            points=_edge_points(routes, coords, s, t),
            label=_label(e, style.edge_label_key) if style.edge_label_key else None,
            style=GlyphStyle(style.edge_layer, stroke=s if style.ink is EdgeInk.SOURCE else style.stroke_index, width=style.edge_width),
            weight=_opt(e, "weight").default_value(0.0),
            caps=(_cap_of(e, "source_cardinality", style.caps[0]), _cap_of(e, "target_cardinality", style.caps[1])),
            route=route,
            source_port=source_port,
            target_port=target_port,
        ))
        for s, t in graph.edge_list()
        for e in (graph.get_edge_data(s, t),)
        for source_port, target_port in (_edge_seat_refs(e),)
    )
    return (*nodes, *edges)


def _seated(ports: tuple[Port, ...], index: int, seats: SeatMap, /) -> tuple[Port, ...]:
    # engine-returned port coordinates read back into `Port.at`, so a rendered seat and a routed endpoint share one
    # truth; off the ELK arm the map is empty and the declared side/index derivation stands.
    return tuple(structs.replace(port, at=seats.get((index, port.id), port.at)) for port in ports)


def _edge_seat_refs(data: object, /) -> tuple[str | None, str | None]:
    # admission already proved each present reference against its endpoint node's declared port ids.
    refs = (data.get("source_port"), data.get("target_port")) if isinstance(data, dict) else (None, None)
    return tuple(ref if isinstance(ref, str) else None for ref in refs)


def _sized(style: EmitStyle, data: dict[str, object], /) -> tuple[float, float]:
    match style.sizing:
        case NodeSizing.FIXED:
            return (style.node_w, style.node_h)
        case NodeSizing.AREA:  # area-true: drawn side tracks sqrt(area); admission proved the column
            side = sqrt(_num(data, "area")) * _AREA_SIDE
            return (side, side)
        case NodeSizing.ENTITY:
            return (style.node_w * 1.6, style.node_h * 2.0)
        case _ as unreachable:
            assert_never(unreachable)


def _label(data: dict[str, object], key: str, /) -> str | None:
    return None if (value := data.get(key)) is None else str(value)


def _keyable(value: object, /) -> bool:
    # the one node-identity admission predicate: a str or int (bool excluded) is the only identity the hash-keyed
    # indexes admit, so every membership test and dict build reads proven-hashable keys alone.
    return isinstance(value, str | int) and not isinstance(value, bool)


def _num(data: dict[str, object], key: str, /) -> float:
    # total read of an admission-proven required column; never called on an optional axis.
    return float(data[key])


def _opt(data: dict[str, object], key: str, /) -> Option[float]:
    # optional numeric axis: absent, per-row null, or a boolean masquerading as int stays Nothing — never a
    # 0.0/1.0 minted from a flag pretending to be a measurement.
    return Some(float(value)) if isinstance(value := data.get(key), int | float) and not isinstance(value, bool) else Nothing


def _shape_of(data: dict[str, object], /) -> NodeShape:
    # node silhouette from the optional `shape` attribute; absent -> RECTANGLE.
    return _NODE_SHAPES.try_find(value).default_value(NodeShape.RECTANGLE) if isinstance(value := data.get("shape"), str) else NodeShape.RECTANGLE


def _sublayout_of(data: dict[str, object], /) -> SubLayout:
    # inner-engine override from the optional `sublayout` attribute; absent -> INHERIT.
    return _sub_member(value) if isinstance(value := data.get("sublayout"), str) else SubLayout.INHERIT


def _sub_member(value: str, /) -> SubLayout:
    return SubLayout(value) if value in {member.value for member in SubLayout} else SubLayout.INHERIT


def _cap_of(data: dict[str, object], key: str, default: EndCap, /) -> EndCap:
    # per-edge ER cardinality wins by value; absent or foreign -> the row's default cap.
    return _CARDINALITY.try_find(value).default_value(default) if isinstance(value := data.get(key), str) else default


def _ports_of(data: dict[str, object], /) -> tuple[Port, ...]:
    # typed connection ports from the optional `ports` attribute; side-local index is the list position, an explicit coordinate seats FIXED_POS.
    return tuple(_port_of(entry, index) for index, entry in enumerate(raw)) if isinstance(raw := data.get("ports"), list | tuple) else ()


def _port_of(entry: object, index: int, /) -> Port:
    match entry:
        case (str() as pid, str() as side, (float() | int() as ax, float() | int() as ay)):
            return Port(pid, _side_member(side), index, at=(float(ax), float(ay)))
        case (str() as pid, str() as side):
            return Port(pid, _side_member(side), index)
        case str() as pid:
            return Port(pid, PortSide.EAST, index)
        case _:
            return Port(f"p{index}", PortSide.EAST, index)


def _side_member(value: str, /) -> PortSide:
    return _PORT_SIDES.try_find(value).default_value(PortSide.EAST)


def _parent_of(data: dict[str, object], /) -> object:
    # compound-nesting parent from the optional `parent` id; absent -> None (a root).
    return None if isinstance(value := data.get("parent"), bool) else value


def _pt(point: tuple[float, float], /) -> Point:
    return (float(point[0]), float(point[1]))


def _oriented(point: tuple[float, float], swap: bool, /) -> Point:
    return (float(point[1]), float(point[0])) if swap else (float(point[0]), float(point[1]))


def _edge_points(routes: RouteMap, coords: LayoutMap, source: int, target: int, /) -> tuple[Point, ...]:
    return routes.get((source, target), (coords[source], coords[target]))


def _proj_scale(policy: LayoutPolicy, /) -> float:
    match policy:
        case LayoutPolicy(tag="projected", projected=(_, scale)):
            return scale
        case _:
            return 1.0


# --- [AREA_LAW] ---------------------------------------------------------------------------
def _floor_total(graph: rx.PyDiGraph, children: tuple[int, ...], /) -> float:
    # floor band's plan-area total over its program children; band length derives from real area.
    return sum(_num(graph.get_node_data(child), "area") for child in children)


def _segments(graph: rx.PyDiGraph, floor: int, children: tuple[int, ...], y: float, /):
    # per-program band segments: cumulative x offsets proportional to the typed `area` columns.
    x = 0.0
    for child in children:  # Exemption: the cumulative x offset is an ordered running fold the generator carries
        width = _num(graph.get_node_data(child), "area") * _AREA_UNIT  # measured area alone sizes the segment — no floor clamp distorting proportion
        yield DiagramGlyph(node=NodeMark(
            index=child, x=x, y=y, w=width, h=_BAND_H,
            label=_label(graph.get_node_data(child), "program"), style=GlyphStyle("program", fill=child), parent=floor,
        ))
        x += width


def _polygon(data: dict[str, object], key: str, scale: float, index: int, label: str | None, layer: str, /) -> DiagramGlyph | None:
    # typed plan-polygon ingress: (east, north) vertices computed upstream, same transform as PARCEL_GRID; shoelace magnitude on the raw ring.
    raw = data.get(key)
    if not isinstance(raw, list | tuple) or len(raw) < 3:
        return None
    ring = tuple((float(east) * scale, -float(north) * scale) for east, north in raw)
    return DiagramGlyph(area=AreaMark(index=index, ring=ring, label=label, style=GlyphStyle(layer, fill=index), magnitude=_shoelace(raw)))


def _shoelace(raw: "list | tuple", /) -> float:
    pairs = tuple((float(east), float(north)) for east, north in raw)
    return abs(sum(x0 * y1 - x1 * y0 for (x0, y0), (x1, y1) in pairwise((*pairs, pairs[0])))) / 2.0


def _sun_furniture(graph: rx.PyDiGraph, radius: float, /) -> Result[tuple[DiagramGlyph, ...], LayoutFault]:
    # composes the solar furniture when the frame carries site columns; an angles-only frame draws bare marks, never fabricated furniture.
    head = next(
        (
            data
            for i in graph.node_indices()
            if isinstance(data := graph.get_node_data(i), dict) and _opt(data, "latitude").is_some()
        ),
        None,
    )
    if head is None:
        return Ok(())
    site = Site(
        latitude=_num(head, "latitude"),
        longitude=_num(head, "longitude"),
        altitude=_opt(head, "site_altitude").default_value(0.0),
        tz=str(head["timezone"]),
        pressure=_opt(head, "pressure").to_optional(),
        temperature=_opt(head, "temperature").default_value(12.0),
    )
    return furniture(site, SolarProjection.STEREOGRAPHIC, radius, int(_num(head, "year"))).map(
        lambda rows: tuple(
            DiagramGlyph(fragment=FragmentMark(d=row.fragment, label=row.label or None, anchor=row.anchor, style=GlyphStyle("furniture")))
            for row in rows
            if row.fragment
        )
    ).map_error(
        lambda fault: LayoutFault(solar=(fault.tag, str(fault)))
    )


# --- [PROJECTIONS] ----------------------------------------------------------------------
def _solar_arc(data: dict[str, object], scale: float, /) -> Point:
    # SAME projection fold that positions the furniture, so a mark and its backdrop cannot disagree.
    return project(_num(data, "azimuth"), _num(data, "altitude"), SolarProjection.STEREOGRAPHIC, scale)


def _floor_band(data: dict[str, object], scale: float, /) -> Point:
    return (0.0, _opt(data, "level").default_value(0.0) * scale)  # model-space stacking band; the sheet viewport flips vertical


def _parcel_grid(data: dict[str, object], scale: float, /) -> Point:
    return (_num(data, "east") * scale, -_num(data, "north") * scale)


def _shells(graph: rx.PyDiGraph, /) -> list[list[int]] | None:
    # `shell` is the radial partition; `ring` remains site-polygon evidence and never shares this scalar identity regime.
    banded = sorted((_opt(graph.get_node_data(i), "shell").default_value(0.0), i) for i in graph.node_indices())
    return [[node for _band, node in run] for _ring, run in groupby(banded, key=itemgetter(0))] or None


PROJECTION: Final[Map[ProjectionKind, Projection]] = Map.of_seq([
    (ProjectionKind.SOLAR_ARC, _solar_arc),
    (ProjectionKind.FLOOR_BAND, _floor_band),
    (ProjectionKind.PARCEL_GRID, _parcel_grid),
])
_FORCE: Final[Map[ForceMode, Callable[[rx.PyDiGraph, int, int], rx.Pos2DMapping]]] = Map.of_seq([
    (ForceMode.SPRING, lambda graph, seed, iterations: rx.spring_layout(graph, seed=seed, num_iter=iterations)),
    (ForceMode.KAMADA_KAWAI, lambda graph, _seed, _iterations: rx.kamada_kawai_layout(graph)),
])
_RING: Final[Map[RingMode, Callable[[rx.PyDiGraph, float], rx.Pos2DMapping]]] = Map.of_seq([
    (RingMode.CIRCULAR, lambda graph, scale: rx.circular_layout(graph, scale=scale)),
    (RingMode.SHELL, lambda graph, scale: rx.shell_layout(graph, nlist=_shells(graph), scale=scale)),
    (RingMode.SPIRAL, lambda graph, scale: rx.spiral_layout(graph, scale=scale)),
])


# --- [ENGINES] --------------------------------------------------------------------------
def _layered(graph: rx.PyDiGraph, direction: LayerDirection, gap: float = _LAYER_GAP) -> LayoutMap:
    return {
        node: _layer_point(depth, rank, direction, gap)
        for depth, layer in enumerate(rx.topological_generations(graph))
        for rank, node in enumerate(layer)
    }


def _layer_point(depth: int, rank: int, direction: LayerDirection, gap: float, /) -> Point:
    match direction:
        case "TB":
            return (rank * gap, depth * gap)
        case "BT":
            return (rank * gap, -depth * gap)
        case "LR":
            return (depth * gap, rank * gap)
        case "RL":
            return (-depth * gap, rank * gap)
        case _ as unreachable:
            assert_never(unreachable)


def _grandalf_layout(graph: rx.PyDiGraph, policy: LayeredPolicy, /) -> LayoutResult:
    # every Graph component lays out (a lone .C[0] read strands sibling components on unwritten views); grandalf ranks grow
    # downward only, so the _RANKING row supplies the BT sign flip and the LR/RL axis swap the provider does not own.
    vertices = {i: GrandalfVertex(i) for i in graph.node_indices()}
    if not vertices:
        return {}, {}, {}
    edges = tuple(GrandalfEdge(vertices[source], vertices[target]) for source, target in graph.edge_list())
    for vertex in vertices.values():  # Exemption: grandalf VertexViewer/EdgeViewer are mutable view sinks the provider reads before draw()
        vertex.view = VertexViewer(_NODE_W, _NODE_H)
    for edge in edges:
        edge.view = EdgeViewer()
    ranking, swap = _RANKING[policy.direction]
    sign = -1.0 if ranking == "up" else 1.0
    turned = lambda x, y, dx: _oriented((float(x) + dx, sign * float(y)), swap)
    offset, coords, routes = 0.0, {}, {}
    for component in GrandalfGraph(list(vertices.values()), list(edges)).C:  # Exemption: per-component draw threads a running x offset
        layout = SugiyamaLayout(component)
        layout.route_edge = _grandalf_router(policy.route)
        layout.init_all()
        layout.draw()
        raw = {int(vertex.data): (float(vertex.view.xy[0]), float(vertex.view.xy[1])) for vertex in component.sV}
        base = min(x for x, _y in raw.values())
        coords |= {index: turned(x, y, offset - base) for index, (x, y) in raw.items()}
        routes |= {
            (int(edge.v[0].data), int(edge.v[1].data)): tuple(turned(x, y, offset - base) for x, y in edge.view._pts)
            for edge in component.sE
        }
        offset += max(x for x, _y in raw.values()) - base + _LAYER_GAP
    return coords, routes, {}


def _grandalf_router(route: EdgeRoute, /) -> Router:
    match route:
        case EdgeRoute.LINES:
            return route_with_lines
        # grandalf ships no orthogonal router; ORTHOGONAL is rewired to ELK upstream, so this arm only ever binds a curve.
        case EdgeRoute.SPLINES | EdgeRoute.ORTHOGONAL:
            return route_with_splines
        case _ as unreachable:
            assert_never(unreachable)


def _sugiyama_layout(graph: rx.PyDiGraph, policy: LayeredPolicy, /) -> LayoutResult:
    ranking, swap = _RANKING[policy.direction]
    arranged = from_edges(
        list(graph.edge_list()), dummy_vertices=True, ranking_type=ranking, crossing_minimization=policy.crossing, vertex_spacing=_LAYER_GAP
    ).dot_layout()
    placed: LayoutMap = {int(node): _oriented(point, swap) for component in arranged for node, point in component[0]}
    real = frozenset(graph.node_indices())
    coords: LayoutMap = {index: placed[index] for index in real if index in placed}
    coords |= _stranded(tuple(index for index in real if index not in placed), coords)  # from_edges drops isolated nodes
    segments = tuple((int(source), int(target)) for component in arranged for source, target in (component[3] or ()))
    return coords, _sugiyama_routes(segments, placed, real), {}


def _sugiyama_routes(segments: tuple[tuple[int, int], ...], placed: LayoutMap, real: frozenset[int], /) -> RouteMap:
    successor = {source: target for source, target in segments if source not in real}  # each dummy routes one edge
    routes: RouteMap = {}
    for source, hop in segments:  # Exemption: a long edge decomposes into a single-successor dummy chain the trace follows to its real terminal
        if source in real and hop not in real:
            chain, node = [source, hop], hop
            while node not in real:
                node = successor[node]
                chain.append(node)
            routes[(source, node)] = tuple(placed[step] for step in chain)
    return routes


def _stranded(absent: tuple[int, ...], placed: LayoutMap, /) -> LayoutMap:
    base = max((y for _x, y in placed.values()), default=0.0) + _LAYER_GAP
    return {node: (rank * _LAYER_GAP, base) for rank, node in enumerate(absent)}


def _elk_layout(graph: rx.PyDiGraph, policy: LayeredPolicy, /) -> LayoutResult:
    document = _elk_document(graph, policy)
    validate_graph(document)  # InvalidGraphException (an ElkError) on a malformed doc, trapped at the offload boundary
    result = ELK().layout(document, layout_options=document["layoutOptions"])  # hierarchyHandling rides the doc root when nested
    coords: LayoutMap = {}
    seats: SeatMap = {}
    _absolutize(result.get("children") or [], (0.0, 0.0), coords, seats)
    # a port-bound edge carries the PORT id in `sources`/`targets`, so the node pair recovers from the edge ordinal, never `int(...)` on an endpoint id
    pairs = {f"e{ordinal}": (int(source), int(target)) for ordinal, (source, target) in enumerate(graph.edge_list())}
    routes: RouteMap = {
        pairs[edge["id"]]: _elk_route(section)
        for edge in collect_edges(result)
        if edge["id"] in pairs and "startPoint" in (section := (edge.get("sections") or ({},))[0])
    }
    return coords, routes, seats


def _elk_document(graph: rx.PyDiGraph, policy: LayeredPolicy, /) -> dict:
    # foreign ELK-JSON contract, built structurally never an f-string splice; node id is the stringified stable index.
    # Every edge stays at the root so its section geometry reads back root-absolute; INCLUDE_CHILDREN lets a cross-container edge route.
    roots, kids = _nest(graph)
    options = dict(_ELK_OPTIONS.items()) | {"elk.algorithm": "layered", "elk.direction": _ELK_DIRECTION[policy.direction]}
    return {
        "id": "root",
        "layoutOptions": options | ({"elk.hierarchyHandling": "INCLUDE_CHILDREN"} if kids else {}),
        "children": [_elk_node(index, graph, kids) for index in roots],
        # an endpoint binds its admitted named port id when the edge declares one, so routes constrain to seats.
        "edges": [
            # explicit None checks: an admitted port id binds even when empty — truthiness would silently rebind it to the node
            {
                "id": f"e{ordinal}",
                "sources": [str(source) if source_port is None else source_port],
                "targets": [str(target) if target_port is None else target_port],
            }
            for ordinal, (source, target) in enumerate(graph.edge_list())
            for source_port, target_port in (_edge_seat_refs(graph.get_edge_data(source, target)),)
        ],
    }


def _elk_node(index: int, graph: rx.PyDiGraph, kids: frozendict[int, tuple[int, ...]], /) -> dict:
    # ports bind under FIXED_ORDER so the layered algorithm seats each on its side/index; a fixed `at` seat escalates the node
    # to FIXED_POS. A `children`-owning node recurses into an enclosed sub-graph, its `sublayout` selecting the container's
    # own `elk.algorithm` when it departs INHERIT.
    data = graph.get_node_data(index)
    base: dict[str, object] = {"id": str(index), "width": _NODE_W, "height": _NODE_H}
    options: dict[str, str] = {}
    if ports := _ports_of(data):
        base["ports"] = [_elk_port(port) for port in ports]
        options["elk.portConstraints"] = "FIXED_POS" if any(port.at is not None for port in ports) else "FIXED_ORDER"
    if nested := kids.get(index):
        base["children"] = [_elk_node(child, graph, kids) for child in nested]
        if (sub := _sublayout_of(data)) is not SubLayout.INHERIT:
            options["elk.algorithm"] = _SUB_ALGORITHM[sub]  # per-container inner engine (recursive compound layout)
    if options:
        base["layoutOptions"] = options
    return base


def _elk_port(port: Port, /) -> dict[str, object]:
    entry: dict[str, object] = {
        "id": port.id,
        "width": port.width,
        "height": port.height,
        "layoutOptions": {"port.side": port.side.value.upper(), "port.index": str(port.index)},
    }
    if port.at is not None:  # node-relative fixed seat; the owning node escalates to FIXED_POS
        entry["x"], entry["y"] = port.at
    return entry


def _nest(graph: rx.PyDiGraph, /) -> tuple[tuple[int, ...], frozendict[int, tuple[int, ...]]]:
    # roots and a parent-index -> child-index map from the optional `parent` node-domain-id, resolved through the id->index lookup;
    # no `parent` anywhere -> all roots, empty map, the flat document.
    lookup = {data["id"]: index for index in graph.node_indices() if isinstance(data := graph.get_node_data(index), dict) and "id" in data}
    kids: dict[int, list[int]] = {}
    roots: list[int] = []
    for index in graph.node_indices():
        parent = lookup.get(_parent_of(graph.get_node_data(index)))
        if parent is not None and parent != index:
            kids.setdefault(parent, []).append(index)
        else:
            roots.append(index)
    return tuple(roots), frozendict({parent: tuple(children) for parent, children in kids.items()})


def _absolutize(nodes: list[dict], origin: Point, coords: LayoutMap, seats: SeatMap, /) -> None:
    # accumulate the container origin down the compound-children tree so each parent-relative ELK `x`/`y` becomes absolute;
    # a flat document recurses once at (0, 0). ELK writes `x`/`y` onto every port node-relative, so each port's center
    # harvests into the SeatMap and the emitted marks carry the engine's own seats.
    ox, oy = origin
    for node in nodes:
        absolute = (ox + float(node.get("x", 0.0)), oy + float(node.get("y", 0.0)))
        index = int(node["id"])
        coords[index] = absolute
        for port in node.get("ports") or ():
            seats[(index, str(port["id"]))] = (
                float(port.get("x", 0.0)) + float(port.get("width", 0.0)) / 2.0,
                float(port.get("y", 0.0)) + float(port.get("height", 0.0)) / 2.0,
            )
        if nested := node.get("children"):
            _absolutize(nested, absolute, coords, seats)


def _elk_route(section: dict, /) -> tuple[Point, ...]:
    bends = tuple(_elk_point(point) for point in section.get("bendPoints") or ())
    return (_elk_point(section["startPoint"]), *bends, _elk_point(section["endPoint"]))


def _elk_point(point: dict, /) -> Point:
    return (float(point["x"]), float(point["y"]))


def _constrained_layout(graph: rx.PyDiGraph, rules: tuple[LayoutRule, ...], /) -> LayoutResult:
    # PLAN-ANCHORED seeding: typed `east`/`north` seats under a `strong` stay, every other node the circular spread under `weak`.
    solver, variables = Solver(), {index: (Variable(), Variable()) for index in graph.node_indices()}
    spread = rx.circular_layout(graph)  # robust any-graph base for nodes without plan coordinates
    for index, (var_x, var_y) in variables.items():  # Exemption: kiwisolver Solver is a mutable sink (addConstraint mutates in place, returns None)
        east, north = _opt(graph.get_node_data(index), "east"), _opt(graph.get_node_data(index), "north")
        anchored = east.is_some() and north.is_some()
        seed_x, seed_y = (east.default_value(0.0), -north.default_value(0.0)) if anchored else _pt(spread[index])
        band = "strong" if anchored else "weak"
        solver.addConstraint((var_x == seed_x) | band)
        solver.addConstraint((var_y == seed_y) | band)
    for constraint in (built for rule in rules for built in _rule_constraints(rule, variables)):
        solver.addConstraint(constraint)
    solver.updateVariables()
    return {index: (float(var_x.value()), float(var_y.value())) for index, (var_x, var_y) in variables.items()}, {}, {}


def _rule_constraints(rule: LayoutRule, variables: dict[int, tuple[Variable, Variable]], /) -> tuple[Constraint, ...]:
    match rule:
        case LayoutRule(tag="align", align=(nodes, axis, strength)):
            head = variables[nodes[0]][_AXIS[axis]]
            return tuple((variables[node][_AXIS[axis]] == head) | strength for node in nodes[1:])
        case LayoutRule(tag="separate", separate=(before, after, axis, gap, strength)):
            return ((variables[after][_AXIS[axis]] - variables[before][_AXIS[axis]] >= gap) | strength,)
        case LayoutRule(tag="distribute", distribute=(nodes, axis, gap, strength)):
            return tuple((variables[nxt][_AXIS[axis]] - variables[cur][_AXIS[axis]] == gap) | strength for cur, nxt in pairwise(nodes))
        case LayoutRule(tag="anchor", anchor=(node, (at_x, at_y), strength)):
            var_x, var_y = variables[node]
            return ((var_x == at_x) | strength, (var_y == at_y) | strength)
        case LayoutRule(tag="mirror", mirror=(left, right, axis, about, strength)):
            return ((variables[left][_AXIS[axis]] + variables[right][_AXIS[axis]] == 2.0 * about) | strength,)
        case _ as unreachable:
            assert_never(unreachable)


_ENGINE: Final[Map[HierarchyEngine, Callable[[rx.PyDiGraph, LayeredPolicy], LayoutResult]]] = Map.of_seq([
    (HierarchyEngine.RUSTWORKX, lambda graph, policy: (_layered(graph, policy.direction), {}, {})),
