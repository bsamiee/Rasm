# [RASM_RHINO_API_ETO_FORMS]

Catalog scope: the `Eto.Forms` construction surface — the control roster, layout containers, windows/dialogs, and menu/toolbar/command families a generator-shaped UI layer owns as rows.

[NAMESPACES]:
- `Eto.Forms` controls — `TextBox`, `TextArea`, `CheckBox`, `NumericStepper`, `DropDown`, `ColorPicker`, `Slider`, `DateTimePicker`, `FilePicker`, `Label`, `Button`, `GroupBox`, `Expander`, `Splitter`/`SplitterFixedPanel`, `Scrollable`, `DocumentControl`/`DocumentPage`, `WebView`, `GridView`/`TreeGridView`/`GridColumn`/`GridItem`/`TreeGridItem`/`ITreeGridItem`, `TextBoxCell`/`CheckBoxCell`/`CustomCell`, `Panel`, `Drawable`, `Control` (lifecycle/focus/mouse/key/drag event families, `DoDragDrop`).
- `Eto.Forms` layout — `StackLayout`, `DynamicLayout`, `TableLayout`/`TableRow`/`TableCell`, `PixelLayout`, `Orientation`, `Padding`.
- `Eto.Forms` windows — `Window` (state/location/logical-pixel-size events), `Form`, `FloatingForm`, `Dialog<T>` (positive/negative/default/abort buttons, `ShowModal`), `DialogResult`, `WindowStyle`, `DialogDisplayMode`, `MessageBox` families, `OpenFileDialog`/`SaveFileDialog`/`SelectFolderDialog`/`FileDialog`/`FileFilter`, `ColorDialog`, `FontDialog`.
- `Eto.Forms` menus and commands — `MenuBar`, `MenuItem`/`ButtonMenuItem`/`CheckMenuItem`/`RadioMenuItem`/`SeparatorMenuItem`/`SubMenuItem`, `ContextMenu`, `ToolBar`, `ToolItem`/`CheckToolItem`/`SeparatorToolItem`, `Command`/`CheckCommand`/`RadioCommand`.
