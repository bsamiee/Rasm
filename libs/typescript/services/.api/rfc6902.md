# [API_CATALOGUE] rfc6902

`rfc6902` supplies RFC 6902 JSON Patch — typed operation discriminated unions, a patch application function, a diff/patch generator, test operation creator, and a JSON Pointer class — for services that produce or consume `application/json-patch+json` patch documents.

## [04]-[IMPLEMENTATION_LAW]

[PATCH_TOPOLOGY]:
- namespace: `rfc6902`; three sub-modules (`diff`, `patch`, `pointer`) re-exported from the barrel
- `applyPatch` mutates the target object in place; clone before calling when immutable semantics are required
- each entry in the `applyPatch` result array is `null` on success or an error instance for that operation index
- `createPatch` produces a minimal Levenshtein-optimal sequence; pass a custom `diff` function to override subtree comparison

[LOCAL_ADMISSION]:
- `MissingError`, `TestError`, and `InvalidOperationError` are discriminable by `instanceof`; pattern-match the result array to route failures.
- `TestError` carries `.actual` and `.expected` for auditing value mismatches.
- `Options.implicitArrayCreation` enables `add "/-"` to create an array when the parent property is absent.

[RAIL_LAW]:
- Package: `rfc6902`
- Owns: RFC 6902 JSON Patch operations, application, and generation; RFC 6901 JSON Pointer
- Accept: plain JSON-serializable objects
- Reject: hand-rolled patch merge or pointer navigation
