/* circle: accent flash, settle to ok with the selected curve */
.task.done .ring { animation: settle 380ms ELASTIC forwards }
@keyframes settle {
  0%   { transform: scale(.8);  background: var(--accent) }
  55%  { transform: scale(1.18) }
  100% { transform: scale(1);   background: var(--ok) }
}
/* checkmark draws via stroke-dashoffset, 80ms delay */
.ring path { stroke-dasharray: 20; stroke-dashoffset: 20;
  transition: stroke-dashoffset 220ms ELASTIC 80ms }
.task.done .ring path { stroke-dashoffset: 0 }
/* strikethrough grows left to right at 120ms */
.label::after { width: 0; transition: width 240ms ELASTIC 120ms }
.task.done .label::after { width: 100% }

What you are feeling

Drop indicator snaps to the nearest gap, never the raw cursor; it moves only when the pointer crosses a row midpoint, so the target reads decisive, not jittery.
The dragged row stays in place at 35% opacity with a 2° tilt; keeping the ghost in the list preserves the sense of origin and the tilt reads as lifted.
The grip dots are the affordance, the whole row is the handle; the target stays large while the gesture stays teachable.
No auto-scroll and no drop animation — left out so the core feel is judged bare; say the word and they join the next pass.
