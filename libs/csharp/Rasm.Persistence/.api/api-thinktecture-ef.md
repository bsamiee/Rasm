# [RASM_PERSISTENCE_API_THINKTECTURE_EF]

`Thinktecture.Runtime.Extensions.EntityFrameworkCore10` supplies EF Core 10 value-converter
registration for Smart Enums and keyed Value Objects generated by Thinktecture source generators,
wiring them into the EF model without manual per-property converter declarations.

## [4]-[IMPLEMENTATION_LAW]

[EF_TOPOLOGY]:
- 36 types across 7 namespaces; the public surface is exclusively extension methods in the `Thinktecture` namespace
- `Configuration` is an internal options class with `UseConstructorForRead` (bool) and max-length-from-strategy policy
- `ModelBuilder.AddThinktectureValueConverters` skips properties that already have a value provider
- `DbContextOptionsBuilder.UseThinktectureValueConverters` registers `ThinktectureDbContextOptionsExtension`, which installs a model-building convention
- `PropertyBuilder.HasThinktectureValueConverter` calls `ThinktectureValueConverterFactory.Create` to resolve the correct `ValueConverter` for the property type
- `ApplyMaxLengthFromStrategy` is called on the `MutableItem` wrapper when using the configuration-based property overload

[LOCAL_ADMISSION]:
- The preferred registration path is `DbContextOptionsBuilder.UseThinktectureValueConverters()` in `OnConfiguring`; it applies automatically to all entity properties.
- `ModelBuilder.AddThinktectureValueConverters()` in `OnModelCreating` is the per-context fallback when DI-based options configuration is unavailable.
- Per-property `HasThinktectureValueConverter` is reserved for types that the convention cannot resolve automatically.
- Properties decorated with an explicit value provider are always skipped by the bulk registration methods.

[RAIL_LAW]:
- Package: `Thinktecture.Runtime.Extensions.EntityFrameworkCore10`
- Owns: EF Core value-converter registration for Thinktecture Smart Enums and keyed Value Objects
- Accept: `DbContextOptionsBuilder.UseThinktectureValueConverters()` as the default registration path
- Reject: manual `HasConversion` calls for Thinktecture types, duplicate converter registration
