# [PY_ARTIFACTS_FORMAT]

The declarative one-context-to-many-formats template/format engine binding a single structured `TemplateContext` into a closed set of output formats. `TemplatePipeline` is ONE owner discriminating output format over the `TemplateTarget` closed vocabulary AND modality over `TemplateTarget | Iterable[TemplateTarget]` — singular renders one format, plural fans the SAME bound context across many formats inside one `anyio` failure boundary — never a per-format pipeline class and never a sheet-set helper beside the singular path. Each format is a row that either DELEGATES to the existing binding owner (the DOCX/PDF/PPTX template-bind arms and the XML/YAML/TOML structured-data arms reach a `document/emit#DOCUMENT` `DocumentPlan` `DocumentMode`; the HTML arm reaches a `document/report#REPORT` `ReportPlan` `ReportKind.TEMPLATE`) or composes the ONE binding no sibling owns (the ODF arm authors odfpy `OpenDocumentText` `.odt` on the cp315 core off the loop through `to_thread`, because `document/emit#DOCUMENT` declares no ODF `DocumentMode` and the only corpus odfpy is the `document/lens#LENS` `ODS_READ` read inverse). The pipeline mints no jinja2 `Environment`, no `DocxTemplate`, no typst `Compiler`, no `python-pptx` `Presentation` — it binds the context once and the `TARGETS` row carries it into the owning producer, the ODF arm its sole owned producer.

Raw binding material is admitted EXACTLY ONCE: `TemplatePipeline.of(node, **payload)` validates the keyword material through the closed `TemplatePayload` `TypedDict` and its module-level `_PAYLOAD` `TypeAdapter`, folding the `extra_items=object` heterogeneous render band into the frozen `TemplateContext.bindings` `frozendict` evidence and materializing every declared key (`template`/`figures`/`assets`/`pdf_standards`/`loader`/`trusted`) onto a typed field, so a malformed payload rails as `Error("<invalid-payload>")` and the interior is total over the admitted context — never a `dict[str, object]` bag a binding arm re-coerces and never a raw constructor a caller fills by hand. Every binding returns `RuntimeRail[Keyed]` — a `(ContentKey, tuple[ArtifactReceipt, ...])` pair the rail threads unchanged, never cracked to `.value` and re-wrapped. The plural fan-out threads the per-target pairs through the runtime `faults#FAULT` `traversed` fail-fast reducer into `RuntimeRail[Block[Keyed]]`, so a spec-sheet rendered to DOCX, PDF, and HTML is ONE `TemplateContext` bound three ways under one cancellation scope, its first delegated fault cancelling the siblings.

`TemplatePipeline` is itself the `core/receipt#RECEIPT` `ReceiptContributor` the runtime sees — the delegated `DocumentPlan`/`ReportPlan` it mints inside an arm are EPHEMERAL and never reach the runtime contribute fold, so the pipeline carries the receipt rows for every artifact it can fully evidence: the directly-minted `office` row `ArtifactReceipt.Office(key, len(blob))` for the ODF bytes it produces (the receipt owner's named byte-only `office` mint, one of the closed family's own constructors exactly as `Ok`/`Some` are `Result`/`Option`'s, never a new template-pipeline case) and the `office`/`report` rows the `_html` arm threads back unchanged from the `ReportPlan.render` in-band `Keyed` pair (the report owner returns its receipts in-band precisely so a composing caller carries them rather than dropping them). The emit-delegated `DOCX`/`PDF`/`PPTX`/`XML`/`YAML`/`TOML` arms surface only the content key (`DocumentPlan.produce` returns `RuntimeRail[ContentKey]` with no byte count to re-evidence), so their `Keyed` pair carries an empty receipt tuple and the `document/emit#DOCUMENT` owner is their receipt owner — the pipeline fabricates no half-populated emit receipt it cannot fill.

def _odf_lower(ctx: TemplateContext) -> bytes:
    from odf.draw import Frame, Image
    from odf.opendocument import OpenDocumentText, load
    from odf.style import Style, TextProperties
    from odf.table import CoveredTableCell, Table, TableCell, TableRow
    from odf.teletype import addTextToElement
    from odf.text import H, P, Span

    document = load(ctx.template) if ctx.template else OpenDocumentText()

    def emphasis(run: RunNode, /) -> Style | None:
        # one cached automatic text-style per (weight, rtl) the runs share, registered once on
        # `automaticstyles` and bound to the `Span` by `style:name` — never a per-run style copy.
        if run.weight < 700 and not run.rtl:
            return None
        key = f"run-{run.weight}-{'rtl' if run.rtl else 'ltr'}"
        if (style := document.getStyleByName(key)) is None:
            style = Style(name=key, family="text")
            style.addElement(TextProperties(fontweight="bold" if run.weight >= 700 else "normal", writingmode="rl-tb" if run.rtl else "lr-tb"))
            document.automaticstyles.addElement(style)
        return style

    def inject(holder: object, runs: tuple[RunNode, ...], /) -> None:
        for run in runs:
            if (style := emphasis(run)) is not None:
                span = Span(stylename=style)
                addTextToElement(span, run.text)
                holder.addElement(span)
            else:
                addTextToElement(holder, run.text)

    def attach(node: DocumentNode) -> None:
        match node:
            case SectionNode(level=level, heading=heading, children=body):
                head = H(outlinelevel=level)
                inject(head, heading)
                document.text.addElement(head)
                for child in body:
                    attach(child)
            case BlockNode(runs=runs, children=kids):
                block = P()
                inject(block, runs)
                document.text.addElement(block)
                for child in kids:
                    attach(child)
            case TableNode(rows=rows, spans=spans):
                grid = Table()
                covered = {(span[0] + r, span[1] + c) for span in spans for r in range(span[2]) for c in range(span[3]) if (r, c) != (0, 0)}
                anchored = frozendict({(span[0], span[1]): (span[2], span[3]) for span in spans})
                for r, row in enumerate(rows):
                    line = TableRow()
                    for c, cell in enumerate(row):
                        if (r, c) in covered:
                            line.addElement(CoveredTableCell())
                            continue
                        rows_span, cols_span = anchored.get((r, c), (1, 1))
                        box = TableCell(valuetype="string", numberrowsspanned=rows_span, numbercolumnsspanned=cols_span)
                        text = P()
                        inject(text, _cell_runs(cell))
                        box.addElement(text)
                        line.addElement(box)
                    grid.addElement(line)
                document.text.addElement(grid)
            case FigureNode(asset_key=asset_key, alt=alt, caption=caption, meta=meta):
                holder = P()
                if (asset := ctx.assets.get(asset_key.hex)) is not None:
                    blob, mediatype = asset
                    href = document.addPictureFromString(blob, mediatype)
                    width, height = (meta.bounds[2], meta.bounds[3]) if meta.bounds else (0.0, 0.0)
                    frame = Frame(width=f"{width}pt", height=f"{height}pt", anchortype="as-char")
                    frame.addElement(Image(href=href))
                    holder.addElement(frame)
                inject(holder, caption) if caption else addTextToElement(holder, alt)
                document.text.addElement(holder)
            case _:
                for child in children(node):
                    attach(child)

    attach(ctx.node)
    sink = BytesIO()
    document.write(sink)
    return sink.getvalue()


def _cell_runs(cell: DocumentNode, /) -> tuple[RunNode, ...]:
    return tuple(leaf for leaf in walk(cell) if isinstance(leaf, RunNode))

class TemplatePipeline(Struct, frozen=True):
    context: TemplateContext

    @classmethod
    def of(cls, node: DocumentNode, /, **raw: Unpack[TemplatePayload]) -> Result["TemplatePipeline", TemplateFault]:
        try:
            payload = _PAYLOAD.validate_python(raw)
        except ValidationError as fault:
            fault.add_note(f"<template.payload:{[error['loc'] for error in fault.errors()]}>")
            return Error("<invalid-payload>")
        bindings = frozendict({name: value for name, value in payload.items() if name not in _DECLARED})
        return Ok(cls(context=TemplateContext(
            template=payload["template"],
            node=node,
            bindings=bindings,
            figures=payload.get("figures", ()),
            assets=payload.get("assets", frozendict()),
            pdf_standards=payload.get("pdf_standards", ()),
            loader=payload.get("loader", ReportLoader.DICT),
            trusted=payload.get("trusted", True),
        )))

    @overload
    async def bound(self, target: TemplateTarget, /) -> RuntimeRail[Keyed]: ...
    @overload
    async def bound(self, target: Iterable[TemplateTarget], /) -> RuntimeRail[Block[Keyed]]: ...
    async def bound(self, target: TemplateTarget | Iterable[TemplateTarget], /) -> RuntimeRail[Keyed] | RuntimeRail[Block[Keyed]]:
        # `TemplateTarget()` is read before the `Iterable` arm because a `StrEnum` is itself
        # iterable; the overloads carry the per-modality output shape so a caller narrows on the
        # target it passes, never re-matching the runtime union — the `traversed` overload idiom.
        match target:
            case TemplateTarget() as one:
                return await self._dispatch(one)
            case stream:
                return await self._fanned(Block.of_seq(stream))

    async def _dispatch(self, target: TemplateTarget, /) -> RuntimeRail[Keyed]:
        arm = _via_document if target in DELEGATES else TARGETS[target]
        return await arm(self, target)

    async def _fanned(self, targets: Block[TemplateTarget], /) -> RuntimeRail[Block[Keyed]]:
        send, receive = create_memory_object_stream[RuntimeRail[Keyed]](max_buffer_size=len(targets) or 1)

        async def fan(target: TemplateTarget, sink: MemoryObjectSendStream[RuntimeRail[Keyed]], /) -> None:
            async with sink:
                await sink.send(await self._dispatch(target))

        async with create_task_group() as group, send:
            for target in targets:
                group.start_soon(fan, target, send.clone())
        return traversed(Block.of_seq([rail async for rail in receive]))

    def contribute(self, outcomes: Block[Keyed], /) -> Iterable[Receipt]:
        for _key, receipts in outcomes:
            for receipt in receipts:
                yield from receipt.contribute()
