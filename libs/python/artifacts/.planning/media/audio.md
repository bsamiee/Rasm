# [PY_ARTIFACTS_MEDIA_AUDIO]

The temporal-artifact AUDIO-stream encode arm — the `EncodeAudio` worker the `media/video#MEDIA` `MediaOp` family dispatches, never a second `Media` owner. `_encode_audio` encodes a `tuple[NDArray[np.int16], ...]` producer sample-block sequence into one single-stream audio container over the FFmpeg floor `av` (PyAV) bundles in its wheel: `av.open(sink, mode="w", format=)` mints the `OutputContainer`, `MediaProfile.voiced` mints the typed `AudioStream`, `OutputContainer.start_encoding` opens the encoder so `AudioCodecContext.frame_size`/`format`/`rate` publish the codec's real values (each reads `0`/unset until the encoder is open), `av.AudioResampler(format=ctx.format.name, layout=profile.layout, rate=ctx.rate)` converts each `s16` producer block to the encoder's required sample format and channel layout, `AudioFifo.write`/`AudioFifo.read(ctx.frame_size, partial=False)` rebuffers the resampled stream to the encoder's exact `frame_size` (AAC `1024`, Opus `960`, FLAC `4608`), and the shared `_drive` `stream.encode(frame)` -> `container.mux(packets)` loop drives the rebuffered chunks with the `stream.encode(None)` flush at end-of-stream, each chunk stamped with its cumulative output-sample `pts` in the `1/rate` audio time base. This page composes the `media/video#MEDIA` container owner — the shared `Media`/`MediaOp`/`MediaProfile`/`MediaEvidence`/`ContainerFormat` family, the `_drive`/`_flush` worker primitives, and the `_SAMPLE_FORMAT` ingest vocabulary — and adds only the `_encode_audio` resample/FIFO/encode worker plus the `MediaProfile.voiced` audio-stream-configure projection it reads; it re-owns no container, profile, evidence, or content-key vocabulary. The `EncodeAudio` case is one `MediaOp` arm dispatched at `media/video#MEDIA`'s `_mux` `match`, worked here by the `_encode_audio` module-level function across the same `anyio.to_process.run_sync` subprocess lane, the muxer discriminated by the typed `ContainerFormat` value the profile carries. The whole encode runs cp315-core in-process: `av` is the abi3-cp311 markerless wheel that covers cp315 with delocate-vendored FFmpeg (no system binary), so the synchronous `start_encoding`/`resample`/`encode`/`mux` C loop is the one CPU-bound body the runtime drives over the subprocess lane, the finished container bytes the only payload crossing back. This arm contributes the single `ArtifactReceipt.Media` case the `media/video#MEDIA` container owner also contributes, never a parallel audio-receipt rail. This page closes the AUDIO half of the `MEDIA` idea.

```python signature
import io

import av
import numpy as np
from numpy.typing import NDArray

from artifacts.media.video import MediaEvidence, MediaProfile, _SAMPLE_FORMAT, _drive, _flush


def _encode_audio(blocks: tuple[NDArray[np.int16], ...], profile: MediaProfile) -> tuple[bytes, MediaEvidence]:
    sink = io.BytesIO()
    frames = samples = 0
    rate = profile.rate
    with av.open(sink, mode="w", format=profile.container.value) as container:
        stream = profile.voiced(container)
        container.start_encoding()  # opens the encoder so codec_context.frame_size/format publish the codec's real values, not 0/unset
        ctx = stream.codec_context
        rate = ctx.rate
        resampler = av.AudioResampler(format=ctx.format.name, layout=profile.layout, rate=ctx.rate)
        fifo = av.AudioFifo()
        for block in blocks:
            source = av.AudioFrame.from_ndarray(block, format=_SAMPLE_FORMAT, layout=profile.layout)
            source.rate = profile.rate
            for converted in resampler.resample(source):
                fifo.write(converted)
                while (chunk := fifo.read(ctx.frame_size, partial=False)) is not None:
                    _drive(container, stream, chunk, samples, ctx.rate)  # pts is the cumulative output-sample offset in the 1/rate base, not the chunk index
                    frames, samples = frames + 1, samples + chunk.samples
        for converted in resampler.resample(None):
            fifo.write(converted)
        if (tail := fifo.read(partial=True)) is not None:
            _drive(container, stream, tail, samples, ctx.rate)
            frames, samples = frames + 1, samples + tail.samples
        _flush(container, stream)
    blob = sink.getvalue()
    return blob, MediaEvidence.measure(profile.container, profile.codec, samples / rate if rate else 0.0, frames, int(profile.bit_rate or 0), blob)
```
