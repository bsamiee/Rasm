# [SNIPPET_GRADING]

Workspace rubric for every `csharp` snippet in this folder's pages. Pages never cite this file. A snippet's score is the MINIMUM across the five axes, stepped at 0.1; the mean is recorded for trend only and never gates anything.

## [1]-[AXES]

### [CARD_ADHERENCE]

| [SCORE] | [ANCHOR]                                                                                              |
| :-----: | :---------------------------------------------------------------------------------------------------- |
|    6    | topic shown, law partial                                                                              |
|    8    | rule and accept proven, boundary implied                                                              |
|    9    | rule, accept, and boundary legible; the rejected spelling's absence is visible                        |
|  9.5+   | every load-bearing clause of the card has a line; the card's advanced interaction is the code's spine |

### [SOPHISTICATION]

| [SCORE] | [ANCHOR]                                                                              |
| :-----: | :------------------------------------------------------------------------------------ |
|    6    | idiomatic but unstacked, or any statement outside a named kernel exemption            |
|    8    | one advanced capability at depth                                                      |
|    9    | stacked capabilities; no flatter spelling visible                                     |
|  9.5+   | deepest operator surface the admitted packages reach for this region; expression-only |

### [DENSITY]

| [SCORE] | [ANCHOR]                                      |
| :-----: | :-------------------------------------------- |
|    6    | no dead lines, ordinary information per line  |
|    8    | 2-3x ordinary                                 |
|    9    | 3-4x; every line decides                      |
|  9.5+   | nothing removable without losing card content |

### [PLACEHOLDER_DISCIPLINE]

| [SCORE] | [ANCHOR]                                                                       |
| :-----: | :----------------------------------------------------------------------------- |
|    6    | one domain-scented identifier                                                  |
|    8    | legal-neutral throughout, minor vocabulary drift                               |
|    9    | perfect corpus-vocabulary neutrality                                           |
|  9.5+   | names teach structure; zero project, host, or domain scent, including literals |

### [FOOTPRINT]

| [SCORE] | [ANCHOR]                                                                  |
| :-----: | :------------------------------------------------------------------------ |
|    6    | inside 15-25 lines with compressible content                              |
|    8    | at most 2 cuttable lines                                                  |
|    9    | lean fixed point                                                          |
|  9.5+   | smallest size that fully captures the card; flat or falling across passes |

## [2]-[AUTOMATIC_FAILS]

An automatic fail overrides every axis score.

| [ID]  | [FAIL]                                    | [ARBITER]                                                                              |
| :---: | :---------------------------------------- | :------------------------------------------------------------------------------------- |
| AF-1  | invented or unverified API                | installed NuGet-cache source; the compile harness once active                          |
| AF-2  | duplicated demonstration                  | `_region-map.md`                                                                       |
| AF-3  | project, host, or domain anchor           | red-team lens RT-2; the orchestrator on adjudication                                   |
| AF-4  | compile failure under the snippet harness | the harness; active from the corpus sweep onward, retroactive over all finalized pages |

## [3]-[BARS]

- 9.2 min-axis: stage admission. Stage prompts and their page-prose grade bars are unchanged by this rubric.
- 9.5 min-axis: finalization — enforced as the stage-12 convergence criterion and re-checked at the corpus sweep.
- Page-prose axes remain at 9.2 throughout.
