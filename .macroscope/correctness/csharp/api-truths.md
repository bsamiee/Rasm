# [CSHARP_API_TRUTHS]

Verified library and host truths a generic reviewer misjudges ‚Äî each listed shape is deliberate or required; flagging it, or demanding its inverse, is a false positive. `LanguageExt.Prelude` is static-imported, so bare `Optional`, `guard`, `Some`, `None`, `unit`, `fun`, `pure`, and `atomic` are legal unqualified; fences use neutral placeholder identifiers (`<value-a>`, `Route`, `Fault`) by law, never as unfinished work.

## [01]-[THINKTECTURE_AND_LANGUAGEEXT]

- `partial` owners are generator-completed, attribute walls grant algebra, `[Union]` closes class-root cases, and `[SmartEnum]` admits no invalid instance. Aggregate closure, cardinality, tolerance, plane, correlation, and case presence stay on the aggregate; member geometry, finiteness, bulges, range, and correspondence stay admitted at the member owner unless a finding cites an admission-bypassing construction path. A zero-initializable value-object keeps one outer default-ghost key seam, and an aggregate checks it on every received row without replaying remaining invariants.
- `using var` on a scoped disposable boundary carrier and `var` bindings inside pattern arms ‚Äî list, property, positional, and slice ‚Äî are doctrine-prescribed spellings; restructure or explicit-type demands against them are style noise at every severity.
- State-threaded `Switch(state: ..., static (s, c) => ...)` collapsing 3+ case overrides into one parent body is the canonical shape; explicit `Switch<TState, TResult>` type arguments and a positional leading state argument are deliberate compiler workarounds ‚Äî never demand per-case virtual methods back or the removal of the type arguments. State-parameter naming defaults to `state` and only `SwitchMapStateParameterName` renames it; arm arity splits by owner ‚Äî `[Union]` arms take `(state, case)`, `[SmartEnum]` arms the state alone ‚Äî and a wrong-arity arm re-binds overload resolution or fails late, a defect regardless of how clean it reads.
- An optional parameter cannot default to a `[ValueObject]`/`[SmartEnum]` value or `default` (CS1736/TTRESG047); `Option<T> x = default` with `IfNone` is the fix and the one optional-parameter form on every rail surface ‚Äî never a smell, while the nullable-defaulting-null parameter is the defect.
- `[Equatable]`/`[OrderedEquality]` (Generator.Equals) on class-root unions is intentional ‚Äî class roots forfeit generated record equality.
- `.As()` re-anchoring after every trait combinator (`TraverseM`, `Traverse`, tuple `.Apply`) is mandatory, not redundant; `.TraverseM` (abort) versus `.Traverse` (accumulate) is a correctness choice, not duplication; traversal lives on the trait carriers alone ‚Äî a bare array or `FrozenSet` owns no `TraverseM`/`Traverse`/`Fold` instance, so the `toSeq(...)` lift at the call head is required, never ceremony.
- `Try.lift(...).Run().MapFail(...).Bind(static r => r)` self-flatten is required ‚Äî dropping the `.Bind` discards the error; `K<F, A>` carrier-polymorphic bodies under trait constraints are one body serving all carriers, never an unresolved generic.
- Operator glyphs are per-carrier algebras (`|` as Validation-choice, Fallible-catch, Schedule-union, or Eff-finally); constrained operators live only inside `extension<E, F, A>(K<F, A>)` blocks ‚Äî the sanctioned mechanism.
- A domain union kept as abstract record with `switch (this)` instead of `[Union]` is a deliberate CS9244 avoidance where a `ref struct` state parameter flows through record consumers.
- An outer lambda parameter renamed from `_` to a real binding where the body uses `_ = expr` is a required CS8820/CS0029 fix ‚Äî never restore the discard convention.
- A block-body lambda copying a mutable struct field to a local before use is required; the IDE0200 method-group reduction silently freezes the receiver by value ‚Äî never accept it.
- Operators without `Negate`/`Multiply` named alternates are canonical (CA2225 is silenced estate-wide); per-type epsilon lives as a typeclass interface member, never one shared constant across differing-unit types.
- Inside a `from..select` expression, `group`/`by`/`into`/`on`/`equals` as range variables or named arguments cause CS0745 cascades; positional arguments and renames are the fix ‚Äî an error wall in a LINQ file is a keyword collision, not a brace imbalance.
- `Schedule` union (`|`) runs to the longer operand, so unioning a finite curve onto an infinite one does not bound it ‚Äî intersect (`&`) or a cumulative-delay gate does; retry wraps each attempt in a fail-only scope so a successful attempt's resources survive to the caller, while repeat wraps every iteration unconditionally so a repeated computation can never acquire and return a live resource.
- `Fin<T>` failure identity is a bare case test ‚Äî every failure equals every other and hashes to one constant ‚Äî so a set or dictionary keyed on `Fin` coalesces all failures into one bucket; observability and dedup key on a projected scalar discriminant first.
- `TraverseM`'s trait-default body delegates to applicative `traverse`, so an un-overridden foldable accumulates under a monadic name; over an effect carrier `Apply` launches both operands while `Bind` is sequential, so `TraverseM` over independent effects serializes them.
- `guard` returns `Guard<Error, Unit>` bound through per-rail `SelectMany`, never a return type; in method position the rail projection (`guard(...).ToFin()`) precedes any combinator ‚Äî a bare `guard(...)` arm or `guard(...).Map(...)` is a compile fiction.
- Implicit `Error` to `Validation<Error, T>` widening fires only in a target-typed position, and generic inference in a `Match` or `Bind` arm never applies a user-defined conversion to unify the arms; a rail-valued lambda spells its fault arm as the concrete carrier explicitly, and `Switch` arms returning sibling case types leave best-common-type inference empty (CS0411) ‚Äî the first arm's cast to the union base is the required anchor, never a redundant cast.
- `default(ImmutableArray<T>)` is a distinct throwing state guarded by `IsDefaultOrEmpty`; `RandomAccess.ReadAsync` returns the bytes actually transferred and never guarantees a full fill ‚Äî read-exactly is the returned-count check against the requested window, and a short return is a malformed-frame fault, never a silent partial.
- `LanguageExt.HashSet` is the immutable persistent carrier ‚Äî update operations return values, `IsEmpty`/`Exists`/folds are its surface ‚Äî so a mutable-BCL reading of it is a false positive, and namespace qualification is the standing recurrence guard.
- `Arr<T>` is a value carrier and cannot hold a null collection reference, so a null-collection demand against it is a false positive.
- `Seq<T>.Head` and `.Last` return `Option<T>` rather than throwing on emptiness, so a throwing-empty-access demand is a false positive.
- Positive relational admission (`value is >= lower and <= upper`) rejects `double.NaN` because both comparisons are false; negative rejection (`value < lower || value > upper`) needs an explicit finiteness fold, so the two forms never share one NaN finding.
- Steps converted to `Validation` before `Traverse` accumulate every failure, `ToFin` following the complete traversal ‚Äî a first-failure short-circuit claim against that shape is false; `Fin`'s `Bind`/`Map`/`TraverseM` evaluate strictly inside a catch lambda, so a host read there is already captured by the surrounding catch frame, and the demanded form pulls every seam host read ‚Äî census and count reads included ‚Äî inside that frame.
- Null is unspellable on a `readonly record struct` parameter and on non-nullable members and union-case payloads under nullable-enable with warnings-as-errors ‚Äî service, delegate, expression-tree, and collection fields included, with `Seq<T>` a readonly struct spelling no null and a host enum-typed field a value type at catalog authority whose zero member is a legitimate roster default at decompile authority ‚Äî a null-admission or malformed-default claim against it misreads the classification, undefined integral values the one representable defect admitted through the defined-value gate; union positional cases construct only over already-admitted payloads, and seam guards ride the host entry weave ‚Äî an interior runtime null-fold demand re-spells the compile contract; host geometry value-struct payloads (`Line`, `Arc`, `Circle`, `Plane`, `Sphere`, `Cylinder`, and kin) spell no null by construction, with only reference-bearing `Curve`, `Surface`, `Mesh`, and `Brep` admitting null at host seams.
- `Enum.IsDefined` is a double admission trap: the non-generic overload throws `ArgumentException` whenever the boxed probe's type mismatches the enum's underlying type ‚Äî an `int` probe over a byte- or short-backed enum compiles and then crashes the happy path ‚Äî and every unnamed `[Flags]` combination reports undefined, so admission rides `Enum.ToObject` or the generic overload with a name round-trip for combinable rosters.

## [02]-[HOST_AND_NATIVE]

- Direct `RhinoApp.InvokeOnUiThread` is the bug ‚Äî it silently swallows exceptions; the `RhinoUi` marshal wrapper is mandatory, never ceremony.
- A host-subclass override of a concrete virtual whose base carries default rendering or behavior (`OnDraw`, `OnPaint`, and kin) chains the base call before its own advice ‚Äî omitting it silently erases the native visual, and a default-only instance renders blank.
- Rhino and GH2 API truths hold at decompile authority, member spellings and lifecycle lineage included: an apparent typo in a decompile-verified host member (`DisplayPipelineAttributes.ShowSilhouttes`, `NamedParametersEventArgs.TryGetUnmanged`) is the shipped spelling ‚Äî a misspelling flag or corrected-spelling demand breaks the binding; obsolete members (`TextFormula`, `SetScale` with `UnitSystem`) are genuinely gone and their `LengthUnit`/rich-text replacements correct; `bool`-returning members where `false` is a benign no-op stay bare-bool, never lifted to `Fin.Fail`; `ObjectList.AllWires` includes dangling entries by design ‚Äî filtering them is the consumer's job, not a library bug; `Canvas.ScheduleRedraw` with the before-background paint phase is the canonical motion tick, and `KeyedValues.Set` is typed-overload-only with no generic `Set<T>`; RhinoCommon display-mode descriptors are owned copies ‚Äî `GetDisplayMode` returns a copy and registration changes publish only through `UpdateDisplayMode`/`AddDisplayMode`, so a mutates-live-registration claim against a copy-then-publish flow is void; a host type's decompile-proven `IDisposable` lineage settles lease-ownership demands ‚Äî custody lands as an owned lease through total union dispatch, so a per-site dispose demand the lease rail already owns, or a custody claim contradicting the lineage, is false; history-replay result wrappers carry internal constructors and update host-owned native state (`ReplayHistoryResult` and kin), so a detached-copy, per-result dispose, or per-result live-object compensation demand against them contradicts the decompiled lineage; an invariant the host surface does not document is never demanded ‚Äî preview-bitmap sizing (`InstanceDefinition.CreatePreviewBitmap` and kin) carries no maximum-dimension contract so the caller's admitted allocation budget governs and a demanded ceiling mints the banned magic number, a host absence sentinel (`Guid.Empty` and kin) collapses to the owned absence case at the read seam and round-trips at apply so a nonempty-invariant demand against that projection fabricates law, and a host member decompile-proven to clamp its declared parameters rather than reject makes pre-host refusal the corpus form, so a host-rejects claim there is void.
- Eto 2.11 macOS workarounds are law, not over-engineering: `using` on Eto brushes is inert, `MeasureString` leaks state (private `FormattedText` instead), `CreateGraphics` throws on Mac, and font-keyed caches add `FontDecoration` because `Font.Equals` omits it.
- Process-static `[ThreadStatic]` caches for expensive native measure/gradient state, and token-gated singleton custody (`Atom<Option<Guid>>` with take/release/withdraw) over native singletons, are required patterns ‚Äî a naive bind/dispose subscription is the bug.
- `TimeProvider` injection (`TimeProvider.System` default) drives motion and clock state; demanding a fake-clock test double the estate deliberately omits is a false positive.
- `SafeHandle` + `[LibraryImport]` + `DangerousAddRef`/`DangerousRelease` + `unsafe` inside a capsule kernel is the named platform-forced exemption, marked `// Exemption:` ‚Äî not unsafe-code smell.
- DuckDB.NET exposes no public zero-copy column span; per-chunk materialization into Arrow builders is the honest pattern ‚Äî demanding zero-copy there is illusory.
- `Complex` lacks `IMultiplyOperators<Complex, double, Complex>`; separate real and complex method bodies are correct, not duplication to unify via generic math.
- Extern aliases (`Triangle` as `TriangleNet`, `SharpVoronoiLib` as `Voronoi`) are preventive collision fixes ‚Äî never demand renames of the colliding domain types or blanket suppressions.
- CsCheck generators filter with `Gen.Where(predicate).Select(transform)`; a throwing `Select` breaks shrinking ‚Äî flag the throw, never "simplify" the pair back into one.
- Admitted-package rulings are settled: QuikGraph stays admitted; MathNet, CSparse, Riok.Mapperly, and Generator.Equals are core substrate to compose at depth, never to wrap.
- Wire-shape and carrier-arity claims settle at the owning constructor: a normalizing constructor folding array and `Seq<T>` into one payload while publishing the array slot shape, and an anchor-row carrier keeping one outer row per query anchor with its complete set within, are contract shapes ‚Äî read the owner before flagging.
- A restore bracket folding the complete facet roster is the page's deterministic-scope contract and the subset fold leaks ambient state ‚Äî never a partial-write defect.
