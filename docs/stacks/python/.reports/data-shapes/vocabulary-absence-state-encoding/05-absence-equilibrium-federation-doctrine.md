# Absence Encoding Equilibrium Federation Doctrine

# Highest-Order Thesis

- Equilibrium law: for every exported `ConceptId`, the system admits one lawful wire‚Üícanonical path (`L`) and one lawful canonical‚Üíwire path (`R`) such that descriptor closure, dominance, semigroup composition, witness erasure, Galois monotonicity, and declared breach tolerance jointly hold ‚Äî violation at any stratum is attributed to the federation row.
- Highest-order invariant: interior domain modules observe only collapsed canonical types and optional comonad audit hash ‚Äî every carrier object (`UnsetType`, `MISSING`, sentinel, `WitnessTag`, `CounitTag`, `PostureClass`, semigroup tables, Galois certificates) stops at named adapter or root federation boundaries; this is the non-negotiable closure criterion.
- Novelty budget is zero new encoding families ‚Äî only federation topology, equilibrium registers, coupling laws between existing snapshots, and orchestrated promotion units; any proposed sixth namespace (`AbsenceAxis` wire token, eighth descriptor bit, merged witness/counit literals) is equilibrium breach.

# Federation Owner Topology

- **Federation owner** ‚Äî one composition-root submodule caches `frozendict[ConceptId, FederationRow]` at import completion; row aggregates read-only handles to `DecisionSnapshot`, `AdjunctionRow`, field tensor slice, and promotion metadata ‚Äî not a merge of dicts at runtime, a frozen product type with single hash.
- **Vocabulary contract import** ‚Äî federation owner imports wire `StrEnum`, sentinel bindings, migration tables, and contract rows from canonical vocabulary owners only; never re-declares tokens or parallel remap dicts.
- **Decision evaluation import** ‚Äî federation owner imports `evaluate`, semigroup table, witness mint/consume, Hasse cover, and handoff `transition_id` keys from the decision-evaluation owner; descriptor basis and fork outcomes remain authoritative there.
- **Adjunction encode import** ‚Äî federation owner imports `L`, `R`, Galois certificate, encode policy, counit mint, and adequacy profile from the adjunction-encode owner; unit/counit metamorphic chains remain authoritative there.
- **Interior ban** ‚Äî domain packages import collapsed type aliases exported by federation owner; import-linter rejects `FederationRow`, semigroup tables, adjunction tables, and both tag namespaces under interior paths.
- **Worker parity** ‚Äî parent, child, and free-threaded worker importers compare full `FederationRow.content_hash` per concept at boot ‚Äî not enum closure, isolated snapshot hash, or per-layer metamorphic pass alone; mismatch attributes to non-deterministic `assemble`, divergent import graphs, or post-import registry mutation racing federation cache publication.

# Namespace Partition Budget

- Allowed meta-vocabularies per concept are exactly seven closed enum namespaces plus two erased tag NewTypes ‚Äî wire `StrEnum`, `Descriptor` basis, `ForkOutcome`, `EncodingDomain`, `WitnessKind`, `PostureClass`, `AbsenceHorizon`; erased `WitnessTag` and `CounitTag` ‚Äî no eighth enum without equilibrium `schema_version` bump and promotion unit that retires a deprecated namespace.

- Literal bands (`Literal`, `T: Literal[...]`) are proof shadows of wire enums ‚Äî federation row links enum primary; literal drift without boundary adapter is vocabulary-owner lattice defect, not federation namespace addition.
- Namespace collision law: identical string value across wire token, witness tag literal, and counit tag literal is `REJECT` ‚Äî federation diff gate enforces triple-namespace injectivity on string values.

# Equilibrium Coupling Laws

- **Snapshot-adjunction lock** ‚Äî `DecisionSnapshot.encoding_domain` must equal domain of `AdjunctionRow.L` output and codomain of `AdjunctionRow.R` input; federation assembly `REJECT` on mismatch ‚Äî neither snapshot nor adjunction row is authoritative alone.
- **Posture-Galois lock** ‚Äî `|PostureClass|` in Galois certificate must equal distinguishability count from descriptor basis under witness-chain cardinality law; federation `REJECT` when `Œ±(descriptor_basis)` and posture fixtures disagree.
- **Witness-counit partition lock** ‚Äî erasure handoff for ingress tags and mint handoff for counit tags must align with `legal_transition_ids` on snapshot; federation row pins exact `transition_id` for each erasure/mint boundary ‚Äî no local stage names.
- **Tensor-adjunction lock** ‚Äî product concepts require independent `Rel` and tensor snapshot slices; coproduct concepts require variant fork on every arm; federation verifies tensor row before adjunction row bind ‚Äî hidden correlation is `TensorEquilibriumFault`.
- **Transitional lock** ‚Äî `ForkOutcome=TRANSITIONAL` on snapshot requires `sunset_instant` UTC and dual adjunction profile on adjunction row; federation assembly fails when sunset present on one row only.
- **Tolerance lock** ‚Äî `adequacy_profile` breach classes on adjunction row must be subset of `breach_classes` on snapshot; relaxing counit without snapshot breach row is `REJECT`.

# Horizon Stage Adjunction Timeline

- Absence evidence flows on three synchronized timelines ‚Äî **horizon** (`CALL`/`SESSION`/`PERSIST`/`AUDIT`), **handoff** (closed `transition_id` slug chain pinned on snapshot), **adjunction phase** (unit path vs counit path vs transitional dual) ‚Äî federation row binds triple `(horizon, transition_id, adjunction_phase)` per concept; desynchronization is `TimelineFault`.
