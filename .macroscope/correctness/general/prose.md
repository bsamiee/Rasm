# [PROSE_LAW]

Durable markdown is agent-facing law governed by `docs/standards/style-guide.md`, `formatting.md`, and `information-structure.md` — findings cite the named defect class and the line. Prose couples to intent (charter, invariant, boundary, ruling, trap); the fence couples to shape; a sentence whose truth depends on the current fence body is drift debt, and a sentence naming the current shape walls the next rebuild inside today's design.

## [01]-[DEFECT_CLASSES]

- Regeneration test governs mirrors: a fact a fresh agent regenerates from disk plus stated invariants is a mirror demanding deletion or demotion to a tooling-verified fence.
- Enumeration anchors — a member roster freezing a census, a forbidden-alternative litany, a frozen count or seal — are findings; the extension rule (how one new row, case, or member lands) is the demanded replacement.
- Hedges (`should`, `might`, `likely`, deferred-decision markers), report frames (session, provenance, freshness, review narration), capability gates (deferral or defense keyed on consumer count), legacy-compat framing, imported upstream posture, and version anchors are findings; contract qualifiers (`optional`, `if present`, `where supported`, `when configured`, `only when`, `unless`) survive, and a prohibition's consequence states in present tense — `would` is the hedge wearing a rationale.
- One owner per fact: a second prose copy at any altitude is a fork waiting to drift; a pointer names the owner and the one consumed symbol, nothing of its internals.
- Near-similar rules accumulated across a passage or file are guidance sediment (docgen SAME_DECISION_SPREAD) demanding the nuance-preserving restructure into one higher-resolution ruling — every distinct decision surviving as its own clause, never a mega-entry; a neighbor separating a different failure class or false-positive discriminator is distinct law, never sediment.
- A fact lands at the lowest tier that owns it: a sentence movable one tier down without losing governing force over more than one child sits at the wrong altitude, and index-tier prose carrying spec-tier mechanism (wire framing, hash algorithms, exact signatures) demotes to the owning spec rather than deletes.
- Routing is a file class, not a section: only `README.md`, `SKILL.md`, `CLAUDE.md`/`AGENTS.md`, and `MEMORY.md` carry links, paths, or cross-references; a relative link or sibling reference in any other durable file is a coupled-link finding, and a page never names itself.
- A nameable surface — member, command, key, flag — is a code span in its exact verified spelling: paraphrased in prose it is a defect, and an exact spelling matching no verifying fence, catalog, or manifest is the fabricated specific — verify or delete, never soften.
- Permission verbs (`supports`, `allows`, `enables`, `provides`, `offers`) where an owning verb states law are findings, including one smuggled into the object of an owning main verb; a package is material the owner consumes, never a benefactor that supplies.
- Deterministic register classes the docgen prose gate's `Check` vocabulary owns — article openers, puffery and copula lexeme rosters, table geometry — are gate territory, never findings.
- A comment carries the one in-situ constraint the code cannot show — the why, the invariant, the trap — never the what, never a duplicate of card or index content, never process or session narration; stacked short fragments merge into one dense line.

## [02]-[STRUCTURE]

- Tables enumerate, cards legislate: a table is earned only when rows share one comparison question with atomic cells; repair is surgical moves in place — hoist repeats to headers, relieve clause tails, split overloaded rows — never conversion to lists because cells run wide.
- A table cell is one atomic unit with no internal comma: a column whose cells all hold one value is tautological and extracts to a card, a column restating its header deletes, and a cell that wants a comma wants to be a card.
- Carrier follows reader job: a repeated paragraph pattern is a record set, a list item with asserted completion is a checkbox, and a command that has not run is an instruction, never a result.
- Raw prose is one physical line per paragraph; a manual newline inside a paragraph is a finding.
- Marker families are closed sets — result, change, lifecycle, state — and invented tokens, emojis, and checkmarks are findings; invocation markers (`[IMPORTANT]`/`[NEVER]`) belong only in instruction and prompt files.
- Fence intent labels (`copy-safe`, `template`, `conceptual`, `signature`, `accepted`/`rejected`, `codemap`, `seams`) bind the body: a body dishonoring its label retags or repairs.

## [03]-[DIAGRAMS]

Mermaid fences answer the diagram template law, never completeness intuition: a strata node earns its place only when a sourcing decision names the member, same-stratum mutual reads ride prose and never edges, a seam graph resolves interior owners home-side only, and an edge-free island rides prose by construction — a finding demanding an excluded node, a same-stratum rung, or an island's edge re-litigates the template. A visual owns one question, is regenerable, and never repeats a table; node and edge labels carry a concept name plus at most one charter phrase, no signatures or versions.
