## [THESIS]

Push every generative kernel and every shared vocabulary DOWN into the foundational plane so no mid- or AEC-plane consumer ever re-derives geometry, color, layer structure, text metrics, or a regime binding a lower owner already mints. ## [ENTRY_CONTRACT] — the `ArtifactWork` law (leg-1 corpus-wide, [V6]/E1)

`ArtifactWork` is `core/plan`'s live node, unchanged in shape:

```python
class ArtifactWork(Struct, frozen=True):
    key: ContentKey                                    # minted over CANONICAL INPUT (spec ⊕ parents), pre-run
    work: Work[ArtifactReceipt]                         # self._emit bound coroutine, UNEVALUATED, threading `key`
    parents: tuple[ContentKey, ...] = ()               # upstream content keys (bound figures, placed sheets)
    admission: Admission = Admission(keyed=None)        # keyed(default) | bare(forced-live) | retried(RetryClass)
    cost: float = 1.0                                   # render-time / byte-volume weight; CPM forward pass
```

**The load-bearing ruling — key over INPUT, not OUTPUT.** Today every producer mints its `ContentKey` over the produced BYTES (`font.engineer` over `_render`, `layered._emit` over `LayerFact.data`), so a key exists only AFTER work runs and elision is impossible (you must produce to know the key). `emit()` instead mints `ArtifactWork.key = ContentIdentity.of(spec_canonical_bytes ⊕ parent_keys, policy=CANONICAL_POLICY)` over the producer's frozen request/spec owner and its parent keys — deterministic and computable before `work` runs — so `core/plan`'s `keyed` admission probes the threaded `cache_seed` and the runtime lane short-circuits BEFORE production (the real in-session elision `core/plan.md` already schedules). `work` threads that same `key` into its terminal `ArtifactReceipt` so `receipt.slot == node.key` (the hit/miss match). A separate output content-address rides an evidence fact only where the store needs it (Gate #3, C#-gated); it is never the elision key. This is the difference between "the engine is fiction" (E1) and a satisfiable pipeline.

**The six carrier conventions collapse to one.** `derive RuntimeRail[ColorReceipt]` / `managed RuntimeRail[ArtifactReceipt]` / `marks·io RuntimeRail[Block[Result[…]]]` / `font RuntimeRail[tuple[ContentKey, ArtifactReceipt]]` / `shape·layout RuntimeRail[ContentKey]` / the `produced`/`rendered`/`_fanned` batch rails — all become `emit(self) -> ArtifactWork | Iterable[ArtifactWork]` whose `work` returns the settled `RuntimeRail[ArtifactReceipt]`. Method names (`derive`/`apply`/`of`/`engineer`/`run`/`lay`/`render`/`produce`/`finish`/`recover`/`author`/`close`) converge on `emit`. `export/layered.emit`/`export/indesign.emit` are the landed proof-of-shape but STILL return `RuntimeRail[ArtifactReceipt]` (the work's terminal, not the node) — they too rewire to return the NODE.

**Multi-artifact arity is law.** A sheet SET or diagram SUITE returns `Iterable[ArtifactWork]` — one node per member with per-member `key`, so a re-issued set re-renders only changed members (per-member elision). A composite artifact (`Register`/`Transmittal`) is ONE aggregate node whose `parents` are its member keys and whose receipt case carries the aggregate facts. Receipts are terminal; incrementality is node granularity, never a partial receipt.

**Constructing owners (core/plan becomes REAL).** `composition/sheet.SheetSet` (the sheet-set issue), `document/report.ReportPlan` (the editorial document package), and `delivery/transmittal.Transmittal` (the ISO 19650 issue) each gather their constituent producers' `emit()` nodes and call `ArtifactPipeline.of(nodes, targets=…)`, then drive through the `runtime/execution#LANE`. `ArtifactPipeline.of` normalizes a lone node or any iterable; `core/plan` is constructed, called, satisfiable — no page's prose may again claim a single production entry it never builds.

**`Admission` selection rule.** `keyed` default (cache-eligible, the elision spine); `bare` for a forced-live non-addressable one-shot; `retried(RetryClass.OCCT)` for a transient-prone offload producer (native render/press/raster/render crossing a worker that can die). The producer picks its case; `ArtifactWork.lowered` `match`es to the lane's `Admit` arm.
