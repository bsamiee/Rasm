# [PY_ARTIFACTS_PLAN]

The content-keyed artifact-production planning layer over the runtime session lane. `ArtifactPipeline` is ONE plan owner that folds a producer dependency graph into the `runtime/execution#LANE` admission shape: each `ArtifactWork` node carries a producer's pre-minted `ContentIdentity`-derived `ContentKey` and its `Work[ArtifactReceipt]` coroutine, the `graphlib.TopologicalSorter` resolves the dependency-level fronts, and every node lowers to one `runtime/execution#LANE` `Admit.keyed = (ContentKey, Work[ArtifactReceipt])` cache unit so a downstream front re-admitting an already-produced artifact replays the upstream `Ok` from the threaded session cache rather than re-rendering. It owns NO cache, NO store, NO scheduler, and NO drain — the `runtime/execution#LANE` `LanePolicy` owns the `CapacityLimiter`/`move_on_after`/`DrainReceipt`, the `runtime/execution#LANE` `StagePlan` owns the `TopologicalSorter` active drive and the front-to-front `DrainReceipt.cache` thread, and the `runtime/execution#LANE` `Admit.keyed` case owns the content short-circuit; `ArtifactPipeline` only PROJECTS the producer graph into those runtime ports and walks the per-artifact `ArtifactReceipt` `contribute` folds into one sub-graph-elision plan. The plan is the third reuse-fabric consumer of the one `receipt/receipt#RECEIPT` `contribute` fold beside the `runtime/execution` lane elision and the `runtime/observability` `MeterProvider` signal stream — it reads each producer's contributed `ArtifactReceipt` as the keyed evidence the elision distinguishes hit from miss on. `ArtifactPipeline.plan` returns a `RuntimeRail[PipelinePlan]` carrying the front-ordered `Block[Admit[ArtifactReceipt]]` lanes keyed by the runtime content key, never an executed result.

```python signature
from collections.abc import Sequence
from graphlib import TopologicalSorter
from types import MappingProxyType
from typing import Final

from expression import Nothing, Option, Some
from expression.collections import Block, Map
from msgspec import Struct

from rasm.runtime.content_identity import ContentKey
from rasm.runtime.faults import RuntimeRail, async_boundary
from rasm.runtime.lanes import Admit, LanePolicy, Work

from artifacts.receipt.receipt import ArtifactReceipt

_DEFAULT_CAPACITY: Final = 8


class ArtifactWork(Struct, frozen=True):
    key: ContentKey
    work: Work[ArtifactReceipt]
    parents: tuple[ContentKey, ...] = ()


class PipelinePlan(Struct, frozen=True):
    fronts: tuple[Block[Admit[ArtifactReceipt]], ...]
    cache_seed: Map[ContentKey, ArtifactReceipt] = Map.empty()


def _topo(nodes: Block[ArtifactWork]) -> TopologicalSorter[ContentKey]:
    order: TopologicalSorter[ContentKey] = TopologicalSorter({node.key: () for node in nodes})
    for node in nodes:
        order.add(node.key, *node.parents)
    return order


def _front_units(front: tuple[ContentKey, ...], by_key: Map[ContentKey, ArtifactWork]) -> Block[Admit[ArtifactReceipt]]:
    return Block.of_seq(
        Admit(keyed=(key, work.work)) for key in front if (work := by_key.try_find(key).default_value(None)) is not None
    )


class ArtifactPipeline(Struct, frozen=True):
    nodes: Block[ArtifactWork]
    capacity: int = _DEFAULT_CAPACITY
    deadline: Option[float] = Nothing
    cache_seed: Map[ContentKey, ArtifactReceipt] = Map.empty()

    @property
    def lane(self) -> LanePolicy:
        return LanePolicy(capacity=self.capacity, deadline=self.deadline)

    async def plan(self) -> RuntimeRail[PipelinePlan]:
        return await async_boundary("pipeline.plan", self._plan)

    async def _plan(self) -> PipelinePlan:
        by_key: Map[ContentKey, ArtifactWork] = Map.of_seq((node.key, node) for node in self.nodes)
        order = _topo(self.nodes)
        order.prepare()
        fronts: list[Block[Admit[ArtifactReceipt]]] = []
        while order.is_active():
            front = order.get_ready()
            fronts.append(_front_units(front, by_key))
            order.done(*front)
        return PipelinePlan(fronts=tuple(fronts), cache_seed=self.cache_seed)

    @staticmethod
    def of(works: Sequence[ArtifactWork], *, capacity: int = _DEFAULT_CAPACITY, deadline: Option[float] = Nothing, warm: Map[ContentKey, ArtifactReceipt] = Map.empty()) -> "ArtifactPipeline":
        return ArtifactPipeline(nodes=Block.of_seq(works), capacity=capacity, deadline=deadline, cache_seed=warm)


LANE_BUDGET: Final[MappingProxyType[str, object]] = MappingProxyType({"capacity": _DEFAULT_CAPACITY, "deadline": Nothing})
```
