## Known smells / partial findings (genuine, unfixed — for a later pass)

- **MCP probes pollute the bridge command history** — the McNeel MCP `run_csharp` and the bridge share the host's command stream, so exploratory `run_csharp` probes leak into `blocks.CommandHistoryRail` evidence (observed: MCP compile-error lines in the command-history tail). Keep the MCP idle during a formal matrix run for clean command-history evidence.
- **startup import cost is structlog-owned (won't-fix)** — `import structlog` pulls `structlog.dev` + the full `rich` tree (~24+35ms) UNCONDITIONALLY; 7+ assay modules import structlog structurally. Deferring `ConsoleRenderer` yields ZERO measurable delta (204.5ms -> 204.6ms); unremovable without forking structlog. PEP 810 `lazy import` is BANNED (ruff `TID254`).
- **SHARED-pull `Artifact.lines=0`** — the SHARED cloud-offload arm reads metadata only, so it leaves `lines=0` (the sftp `TRANSFER` arm populates it). A consumer reading `Artifact.lines` uniformly across strategies must read lazily through the store.
- **Spectral open-mesh receipt semantics** — for open meshes: `PositiveStar1Count < EdgeCount` when a flat interior diagonal is present (open-vs-closed discriminant); `BoundaryCompositionTolerance` defaults to `0.0` on the `EdgeConnection` path that never populates it (a consumer must read `0.0` as "no DEC gate ran"); `SignpostTransportReceiptOf` skips edges with `w < ZeroTolerance` so `TransportedEdgeCount` can under-count `IntrinsicEdgeCount` (compare both, don't assume equality).
- **`RHINO_WIP_APP_PATH` override bypasses the major>=9 filter** — the host-launch exclusion filters only auto-discovery; an explicit override is honored verbatim. The live matrix should assert the leased bundle's `Numeric.Major >= 9` regardless of resolution (live `bridge status` confirmed major 9 this pass).
- **DOC502 vs structured `Raises:`** — ruff rejects a `Raises:` section without an in-body `raise`, so propagate-through-helper exceptions document propagation as prose, not the machine-readable section. Candidate for a repo-wide pydoclint exception.
- **`registry._distill` reaches private `engine._measure`** (composition -> core) — one-hop co-owned seam (`# noqa: PLC2701`), stable; promote to a public engine seam if the `Measurements`/`to_resources` shape forks.
- **`Diagnostic.failing_step` is `Step` in-process, plain `str` after a wire round-trip** — `==`/`!=` safe (StrEnum); identity (`is Step.X`) on decoded envelopes silently fails. The one site uses `!=`; flag if identity is introduced.
- **Static streaming gap** — boundary events stream, but per-phase start/end, per-process receipts, SARIF folds, and discovered-diagnostics emission buffer until the final envelope.
- **`SeamProbe.projected[K]` unused** in the assay specialization — a real engine affordance (post-hoc `calls`-log view); flagged so a reviewer does not strip it.
- **MemoryViewType identity asymmetry** — `st.binary().map(memoryview)` round-trips by equality (msgspec decodes to `bytes`); inert.
- **`_slot_census` dirty read** — reads sibling slot-lock blocks WITHOUT flock (telemetry only); a mid-write block decodes to `None` and undercounts `N` by one. Acceptable for a note; never an admission gate.
- **Whole-program type-check gated by `_empty_route`** — a C#-only or docs-only `static` invocation does not re-check Python; CI must route the full Python tree on merge to close the window.
