Distillation turns each lane's full reservoir in `<WORKSPACE>` — the bedrock and the divergent series above it — into the single source the page is authored from: the cards worth making and the depth each carries. One agent per lane folder.

[DOCTRINE]:
- Every agent — the orchestrator and every distill agent — fully reads `<DOCTRINE>` before any other work: the laws, the collapse scan, the page craft, and what `<TARGET>` owns. That single read is the whole alignment requirement, and every selection, edit, and grade answers to it.

[DISTILL]:
- Read all of `00-bedrock.md` and every divergent file in the lane — the whole reservoir, nothing skipped.
- Select and order the strongest material into one coherent body: the candidate cards and sections for the page, each held at the depth it needs. The divergent series sets the ceiling — the advanced rules, collapses, and integrations lead; the bedrock supplies the load-bearing foundations a candidate still needs, so every entry stands complete on its own.
- Shape each candidate the way the page will consume it: the controlling rule first, then its accept and reject edges, its boundary, and — only where an advanced pattern justifies it — the snippet that proves it.
- Cut what does not earn a card: table-stakes material, weaker duplicates of a stronger candidate, and findings whose value the doctrine's laws cannot place.
- Write the result to `00-distilled.md` in the lane folder.

[DISCIPLINE]:
- Findings and candidate cards only — logical category groups, with no meta-commentary, sourcing, versions, dates, cross-references, or introduction, routing, summary, or validation section. One H1, then `[GROUP_LABEL]:` groups of contiguous bullets with no blank line between items, and a blank line only around a fenced code block, with neutral placeholder names.

[GRADE]:
- The distill agent scores its own `00-distilled.md` one to ten on four axes — richness, veracity, density, advancement — and on relevance to the lane, against a bar of 9.2 on each. Grading is critical, harsh, but fair — not a 9.2 out of convenience, not low for nit-picking. Rework until it clears.
- The main agent then reads each `00-distilled.md` cold — giving no weight to the distill agent's pass — and grades and critiques it the same way, returning anything short, or any line that is not a finding or a strong candidate card, for a targeted fix.

The distillate is the reservoir the page is mined from — more than the page will use, so the best can be chosen.
