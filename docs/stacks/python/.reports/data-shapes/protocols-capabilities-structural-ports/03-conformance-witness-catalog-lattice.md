# Conformance Witness Catalog Lattice

# Critical Signals

- Every admitted `port_symbol` declares one frozen catalog row binding introspection hash, law slugs, evidence grades, bind graph, and CI gates — metamorphic and generative proof consumes row columns as field authority; hand-maintained slug lists beside row refs fail admission.
- Law slugs name reusable proof morphisms — static assignability, frozenset snapshot, runtime smoke, tier traversal, Kleisli bind, re-bind metamorphic — registered per `port_symbol` via `register_law(Port, slug)`, not inferred from pytest file paths.
- Harness execution order pins static checker trinity before `get_protocol_members` frozenset before runtime smoke before semantic `TypeIs` before generative backend coverage before cross-process re-bind — catalog `evidence_grades` column declares subset; `ci_gate_ids` block expensive layers when cheaper layers fail.
- Catalog is proof and CI policy only — interior modules must not import witness tables as runtime routing authority; composition roots register rows; boundary law modules and parametrized harnesses consume them.
- Witness catalog owns row shape, slug band, defect classes, gate bindings, and harness layer map — runtime evidence grades (), Kleisli pipeline laws (), promotion unit atomicity (), and steady-state assurance invariants () cite this wave by foreign key without restating witness tables.

# Witness Row Shape

- Every admitted port declares one frozen catalog row: `port_symbol`, `member_frozenset_hash`, `law_slugs`, `evidence_grades`, `bind_row_id`, `tier_policy`, `async_family`, `consumer_contexts`, `negative_defect_ids`, `ci_gate_ids`.
- `port_symbol` is the unbound protocol class — `Decode`, `ArtifactFileSystem`, `AsyncReadable` — never a parameterized alias (`Decode[int]`) or mirror re-export alias hiding the canonical owner module.
- `member_frozenset_hash` fingerprints `frozenset(get_protocol_members(Port))` at promotion time — unexpected diffs force simultaneous mirror extraction, bind row, and law witness updates in one governance unit.
- `law_slugs` is a closed tuple drawn from the slug band below — orphan ports without at least one slug fail harness discovery; pytest file paths are not valid slug substitutes.
- `evidence_grades` tags which proof layers the port exercises — subset of `{static, frozenset, runtime_smoke, typeis_semantic, generative, metamorphic_rebind}` — ordered per harness execution policy static → frozenset → runtime → generative → metamorphic.
- `bind_row_id` references the composition-root scope registration row — `type[Port]` key, factory symbol, tier traversal table — orphan adapters without bind rows or ports without factories fail integration matrix review.
- `tier_policy` is one of `{none, ordered_short_circuit, version_gated, version_and_tier}` — `none` when single backend; ordered rows name `Block[tuple[Tier, type[Port]]]` hash when multi-backend contexts document primary/secondary/fallback.
- `async_family` is `{sync, async, dual_split}` — `dual_split` mandates separate law slugs for sync and async pipeline shells; bridging through `async def` on sync port methods is indexed negative `sync_async_bridge`.
- `consumer_contexts` lists bounded contexts importing the port through anti-corruption boundary modules — empty when providing context is sole consumer; negative import tests reference this column.
- `negative_defect_ids` index harness anti-patterns each port must reject — `partial_missing_member`, `undecorated_refinement`, `scheduling_skew`, `fat_mirror_unused`, `ambient_registry`.
- `ci_gate_ids` bind merge blockers — `frozenset_drift`, `inherited_runtime_checkability`, `ingress_port_field`, `interior_isinstance`, `vendor_in_all` — to representative matrix modules named in the row metadata.

# Law Slug Band

- `static-assignability` — third-party structural values assign without nominal `class Adapter(Port)` unless signatures genuinely translate; ty, mypy, pyright parity on the same module.
- `member-frozenset` — golden `get_protocol_members` snapshot; CI diffs on unexpected promotion or retirement.
- `runtime-smoke` — `@runtime_checkable` presence with explicit re-decoration on every refinement link used in `isinstance`; `-W error::DeprecationWarning` under GH-132604 policy.
- `typeis-semantic` — named `TypeIs[Port]` predicates when attestation, tier, version band, or non-empty state exceeds PEP 544 presence.
- `default-member` — structural double re-binds protocol defaults — `transaction = Port.transaction` — and omits optional overrides.
- `negative-partial` — one targeted test per defect class from `negative_defect_ids`; mega invalid objects are rejected.
- `tier-traversal` — ordered candidate backends short-circuit on first `Ok(Certified[C])`; weaker backends must not run after acceptance.
- `version-gate-ingress` — `Map[Ver, Callable]` Kleisli chain owns `Unsupported` and `Sunsetted` before interior hops.
- `kleisli-bind` — fault union threading, provenance on contravariant refine hops, `ask(Port)(scope)` resolution precedes method dispatch.
- `fault-severity-routing` — recoverable versus terminal discriminants route at composition seam; interior re-encoding tier tokens is indexed negative `context_leak`.
- `async-shell-fanout` — `@effect.async_result` generator sites collect `Block[T]` or accumulated `Fault` at capability lattice seam.
- `rebind-metamorphic` — shape round-trip and capability re-bind are independent obligations when port-backed egress participates.
- `consumer-negative-import` — consuming contexts cannot import vendor implementation modules; factory-only wiring paths satisfy production graphs.
- `scoped-immutability` — parallel tests prove sibling tasks do not observe transient scope writes from `scoped(caps, thunk)`.

# Defect Class Catalog Rows

- `partial_missing_member` — object lacks one name from `get_protocol_members(Port)`; assert `not isinstance(partial, Port)` when runtime gate is load-bearing; static-only ports omit runtime row.
- `scheduling_skew` — sync port satisfied by `async def` member or inverse; primarily static rejection via assignment helpers; runtime smoke may still pass on name presence alone.
- `parameter_widen_return_narrow` — static-only signature skew; `@pytest.mark.xfail` on typed assignment modules; not witnessed by `@runtime_checkable` alone.
- `undecorated_refinement` — `isinstance(x, Narrow)` against undecorated child of `@runtime_checkable Wide` emits 3.15 deprecation; negative test asserts warning category before 3.20 hard `TypeError`.
- `undecorated_join` — intersection join used in `isinstance` without independent `@runtime_checkable` when join type is the gate subject.
- `sync_async_bridge` — sync port method declared `async def` for blocking IO; static proof primary witness; evolution shortcut forbidden.
- `context_leak` — tier or version vocabulary re-encoded inside interior pipeline after root resolution; attributed to seam projection owner.
- `ambient_registry` — module-level mutable dict replaces explicit `Map[type, object]` scope threading; fails CI gate `ambient_registry`.
- `interior_isinstance` — domain transform re-checks port conformance mid-pipeline; grep gate on non-boundary packages.
- `pickled_capability` — multiprocessing handoff serializes port instance; settings cross IPC; adapters rebuild at worker entrypoint.
- `mock_erased_standin` — `unittest.mock.Mock` satisfies smoke without signature proof; collapses to structural double from `get_protocol_members`.
- `fat_mirror_unused` — mirror declares members no usage graph invokes; trim surface or fail `fat_mirror` CI row.

# CI Gate Row Catalog

- `frozenset_drift` — new required member without updated `member_frozenset_hash`, law slugs, negative partial tests, and bind documentation in same promotion unit.
- `inherited_runtime_checkability` — subclass `isinstance` without explicit `@runtime_checkable` on that link; lint under `-W error::DeprecationWarning`.
- `ingress_port_field` — Pydantic or msgspec model field typed as `Protocol`; capability moves to constructor or operation parameter at root.
- `per_variant_port_sprawl` — three or more protocols differing only by variant arm identity; collapse to one port over `Member` with interior exhaustive `match`.
- `vendor_in_all` — vendor adapter class in public `__all__`; export port symbol and `resolve_*` factory only.
- `interior_isinstance` — non-boundary package grep hits `isinstance` on port symbols; narrowing belongs at root gate or AOP structural prefix.
- `stringly_capability_gate` — authorization compares raw capability strings; frozenset subset algebra on closed `Cap` bands required.
- `checker_parity_gap` — port module green on one backend with suppressions while matrix module fails on others; proof debt tracks on language axis.
- `orphan_port_symbol` — admitted port lacks catalog row, bind row, or `register_law` witness; arch review blocker.
- `law_slug_on_path` — law registration keyed to pytest file path instead of `port_symbol`; harness discovery fails.

# Harness Layer Binding

- Layer 1 `static` executes before Layer 2 `frozenset` before Layer 3 `runtime_smoke` before Layer 4 `typeis_semantic` before Layer 5 `generative` before Layer 6 `metamorphic_rebind` — catalog `evidence_grades` column declares subset; expensive layers block when cheaper layers fail.
- Each `law_slug` maps to exactly one harness layer — `kleisli-bind` and `fault-severity-routing` are Layer 5 generative or integration depending on backend participation; `member-frozenset` is always Layer 2.
- Port law modules co-locate with boundary packages — witnesses import `port_symbol` and adapter factories from providing context; domain modules under test never import vendor backends for catalog convenience.
- Mutation testing targets rows where `law_slugs` includes `tier-traversal`, `kleisli-bind`, or `version-gate-ingress` and seam logic is dense — interior folds on narrowed ports are secondary surfaces.
- Arch import rules extend to catalog consumers — domain folders never import vendor client modules; `consumer-negative-import` slug proves negative paths per `consumer_contexts` column.

# Metamorphic Witness Sub-Rows

- `rebind-metamorphic` parent row splits into sub-rows when only a port operation subset participates — `read_only_observation` omits write members with explicit rationale in law module docstring.
- Sub-row `handoff_stages` lists ordered seam ids — `canonical_materialize`, `interior_port_invoke`, `wire_project`, `decode`, `worker_scope_bind`, `interior_reinvoke` — shape round-trip and capability re-bind remain independent boolean columns.
- `adapter_identity_must_differ` is `true` on every sub-row crossing process seams — parent-process port instances never satisfy worker proof.
- `trusted_replay_pins` optional tuple — encoder module, store key, schema version, factory module — when `trust_posture` on replay chain skips canonical re-validation but not port re-bind.
- Tier metamorphic sub-row varies candidate order in property tests — asserts short-circuit supremum and tier evidence in rejection fault payloads.

# Consumer Seam Witness Rows

- Providing context publishes catalog row with `consumer_contexts` and `bind_row_id` — vendor credential modules, traversal machinery, and tier tables stay private to providing boundary package.
- Consuming context maintains mirror catalog row fragment — `consumer_contexts` self-entry, `consumer-negative-import` slug, imported `port_symbol` only through anti-corruption module path recorded in row metadata.
- Port evolution promotion units appear in changelog rows tied to `member_frozenset_hash` diff and added or retired `law_slugs` — consumers upgrade on catalog diff, not vendor class pins.
- Duplicate mirror ports across two contexts collapse to one `port_symbol` owner — parallel identical mirrors in two `consumer_contexts` lists fail merge review unless anti-corruption remap documents explicit fork with sunset date.
- Cross-context handoff transmits canonical shapes, fold-derived receipts, or immutable configuration records — catalog rows mark `pickled_capability` and `ambient_registry` as hard negatives on consumer importers.

# Collapse And Promotion Rows

- Single-implementer port with `tier_policy=none` and no planned substitution — collapse row candidate when `law_slugs` lacks `generative` and `tier-traversal`; harness reviews concrete boundary parameter unless third-party replacement ships within one release cycle.
- Runtime-only port with `evidence_grades={runtime_smoke}` and no static call sites — collapse to `TypeIs` over nominal owner or delete gate; catalog row retired with explicit collapse `transition_id`.
- Promotion row adds required member — `law_slugs` gains `negative-partial` defect rows per new name, `member_frozenset_hash` rotates, `bind_row_id` factory signatures update, pipeline laws refresh in same unit.
- Optional capability promotion via protocol default member — `default-member` slug required; nominal mixin ABC collapse when defaults preserve third-party structural assignability.
- Async family split promotion — `async_family` transitions from `sync` to `dual_split`; new `async-shell-fanout` slug and async mirror protocol row land without `sync_async_bridge` shortcuts on existing sync port.

# Failure Archaeology Keys

- Catalog attribution routes port-owner-first — static assignability failure targets `static-assignability` row; frozenset diff failure targets `member-frozenset`; deprecation warning targets `undecorated_refinement` or `undecorated_join` defect row.
- `Missing[Port]` at pipeline bind targets `bind_row_id` and `scoped-immutability` slug owner, not domain interior transforms.
- `context_leak` defect attributes to seam projection row when tier or version faults appear mid-interior hop after root resolution.
- Observability spans on seam adapters emit `port_symbol`, `bind_row_id`, tier selected — catalog row metadata supplies stable trace vocabulary; interior spans omit vendor module paths.
- Diagnostic tags on cross-process failures name `rebind`, `scope`, or `ipc` per metamorphic sub-row — undifferentiated adapter errors indicate missing catalog attribution column.

# Representative Catalog Examples

- Minimal single-method port — `law_slugs=("static-assignability", "member-frozenset", "negative-partial")`, `evidence_grades=("static", "frozenset")`, `tier_policy=none`, `async_family=sync`, `negative_defect_ids=("partial_missing_member",)`.
- Generic invariant `Port[T]` — add `runtime-smoke` when `@runtime_checkable` gate is load-bearing; add `typeis-semantic` when attestation predicates exceed presence.
- Intersection join `Dual[T]` — `negative_defect_ids` includes `undecorated_join`; runtime gate names join type requires decoration on join row.
- Multi-backend filesystem port — `tier_policy=ordered_short_circuit`, `law_slugs` includes `tier-traversal`, `Certified[C]` provenance in bind metadata, `tier-traversal` property varies order.
- Versioned codec port — `tier_policy=version_gated`, `law_slugs` includes `version-gate-ingress`, closed `Ver` literal band on companion `Versioned` metadata.
- Egress-participating store port — `evidence_grades` includes `metamorphic_rebind`, `rebind-metamorphic` sub-rows with `handoff_stages` and `adapter_identity_must_differ=true`.

# Bind Row Sub-Schema

- `bind_row_id` resolves to a frozen bind record beside the composition root — `port_key: type[Port]`, `factory_symbol`, `factory_result: Result[Port, ConfigFault]`, `scope_mutation: immutable_fold`, `tier_table_hash` optional.
- `factory_symbol` names `resolve_*` or equivalent narrow factory — consuming contexts import factory plus `port_symbol`; vendor client modules never appear in bind record exports.
- `scope_mutation` describes how `(type(c), c)` pairs enter `Map[type, object]` — `block.fold(lambda s, c: s.add(type(c), c), scope)` pattern; string registry keys are indexed negative `ambient_registry`.
- `tier_table_hash` fingerprints ordered `Block[tuple[Tier, type[Port]]]` when `tier_policy` is not `none` — reordering tiers rotates hash and requires `tier-traversal` law refresh in same promotion unit.
- `ContextVar` absorb wrappers reference bind row only at stack edge — smart constructors and model validators do not call `ask(Port)` from ambient context.
- Orphan `bind_row_id` without matching `port_symbol` catalog row or inverse orphan port without bind record both fail `orphan_port_symbol` CI gate.

# Evidence Grade Alignment Sub-Rows

- `static` sub-row pins checker trinity modules — same port declaration file must pass ty, mypy, and pyright without suppressions before `frozenset` layer executes.
- `frozenset` sub-row stores literal member names beside hash — diff-friendly CI output names added or removed members without re-parsing protocol source.
- `runtime_smoke` sub-row lists each refinement link requiring independent `@runtime_checkable` — link ids map to protocol qualified names in matrix module.
- `typeis_semantic` sub-row names predicate functions and required attestation fields — lambdas are rejected; predicate symbols appear in `typeis-semantic` law module imports.
- `generative` sub-row declares backend brands exercised and hypothesis strategy entrypoints — unfiltered object factories filtered only at runtime are indexed negative `mock_erased_standin`.
- `metamorphic_rebind` sub-row cross-links `handoff_stages` to metamorphic parent slug — omitted stages must document explicit omission rationale in law module header.

# Parametrized Harness Consumption

- Policy runners discover catalog rows by `port_symbol` — `register_law(Port, slug)` slugs must equal `law_slugs` tuple entries; mismatch fails harness registration smoke before backend generative suites start.
- Parametrized pytest modules accept `port_symbol` and `law_slug` — one test function per slug per port; mega-tests covering all slugs on one parametrized axis are rejected for failure localization.
- `negative_defect_ids` drive table-driven negative modules — each id maps to one fixture factory and one assertion family; defect id without fixture row fails catalog completeness review.
- `ci_gate_ids` map to lint or grep jobs with explicit file globs recorded in row metadata — gate id without job binding is prose-only and fails done criteria.
- Integration matrix parametrizes `consumer_contexts` for `consumer-negative-import` — each context runs import graph negative test independently; combined multi-context mega-import test is discouraged.
- Golden `member_frozenset_hash` rotation emits changelog entry with added and retired `law_slugs` — consumers read changelog before upgrading anti-corruption imports.

# Refinement Chain Catalog Linkage

- Refinement chains declare ordered `refinement_link_ids` optional column — `Wide`, `Narrow`, `Narrower` qualified names with per-link `@runtime_checkable` flags when `runtime_smoke` is in `evidence_grades`.
- Child link `member_frozenset_hash` must supersede parent hash on name set — incompatible signature refinements split into separate `port_symbol` rows rather than one chain row with coerced joins.
- `get_protocol_members` golden tests run per link when chain length exceeds one — parent-only frozenset snapshot is insufficient for chain ports.
- Re-decoration promotion on any chain link rotates every downstream link id in `runtime_smoke` sub-row — partial re-decoration updates fail `inherited_runtime_checkability` gate.
- Collapse review retires entire chain row when only terminal link has static call sites — intermediate refinements without gates collapse to single `port_symbol` or `TypeIs` nominal predicate.

# Catalog Registry Shape

- Root `PORT_WITNESS_CATALOG: tuple[WitnessCatalogRow, ...]` extends wave law-registration obligations — every admitted port binds exactly one catalog row plus optional metamorphic and evidence-grade sub-rows; orphan ports without catalog rows fail `orphan_port_symbol` CI gate.
- `WitnessCatalogRow` is a frozen record beside the providing-context boundary package — `port_symbol`, `member_frozenset_hash`, `law_slugs`, `evidence_grades`, `bind_row_id`, `tier_policy`, `async_family`, `consumer_contexts`, `negative_defect_ids`, `ci_gate_ids`, optional `refinement_link_ids`, optional `promotion_unit_id` when row is mid-flight.
- `BindRow` resolves through `bind_row_id` at composition root — `port_key: type[Port]`, `factory_symbol`, `factory_result: Result[Port, ConfigFault]`, `scope_mutation`, optional `tier_table_hash`; catalog row and bind row share `port_symbol` key or both fail orphan gate.
- `MetamorphicSubRow` attaches to parent catalog row when `rebind-metamorphic` is in `law_slugs` — `handoff_stages`, `adapter_identity_must_differ`, optional `trusted_replay_pins`, `read_only_observation` rationale when write members omit.
- `EvidenceGradeSubRow` attaches per grade in `evidence_grades` column — checker module paths, literal member names beside hash, refinement link ids, predicate symbols, hypothesis entrypoints, metamorphic stage cross-links per evidence grade alignment sub-schema.
- Registry build is import gate — bounded contexts without `PORT_WITNESS_CATALOG` row per admitted `port_symbol` fail arch review before parametrized law modules execute.
- Duplicate catalog declarations — test file re-declares `law_slugs` parallel to owner catalog table — collapse to single parametrized import from providing-context registry per steady-state assurance policy.

# Law Slug Harness Layer Map

- `LAW_SLUG_LAYER_MAP: frozendict[str, EvidenceGrade]` is total over the closed slug band — each slug maps to exactly one harness layer; unmapped slug at `register_law` discovery is merge blocker.
- Layer 1 `static` — `static-assignability`, `default-member`, checker trinity parity modules; `parameter_widen_return_narrow` defect witnesses attach as static-only xfail modules without a dedicated slug.
- Layer 2 `frozenset` — `member-frozenset` golden snapshot; literal member name diff beside `member_frozenset_hash`.
- Layer 3 `runtime_smoke` — `runtime-smoke`, `negative-partial` when defect is runtime-gated, `consumer-negative-import` import-graph negatives.
- Layer 4 `typeis_semantic` — `typeis-semantic` named predicate symbols and attestation field requirements.
- Layer 5 `generative` — `tier-traversal`, `version-gate-ingress`, `kleisli-bind`, `fault-severity-routing`, `async-shell-fanout`, `scoped-immutability` when parallel scope or pipeline participation is load-bearing.
- Layer 6 `metamorphic_rebind` — `rebind-metamorphic` parent slug and metamorphic sub-rows with `handoff_stages`.
- Layer map law — expensive layer registration smoke fails when cheaper layer modules for the same `port_symbol` are red; promotion `proof_layers_to_replay` subsets obey same ordering.
- Slug addition extends `LAW_SLUG_LAYER_MAP` in the same promotion unit as catalog `law_slugs` tuple — orphan map entry without catalog row or inverse fails `orphan_port_symbol` and `law_slug_on_path` discovery smoke.

# CI Gate Discovery Rows

- `ci_gate_ids` on each catalog row map to representative matrix modules with explicit file globs — gate id without job binding is prose-only and fails catalog completeness review.
- `frozenset_drift` gate executes on any diff touching `get_protocol_members` — blocks merge until `member_frozenset_hash`, `law_slugs`, negative partial fixtures, and bind documentation rotate in one promotion unit.
- `inherited_runtime_checkability` gate binds to refinement-chain modules under `-W error::DeprecationWarning` — undecorated child or join links fail before runtime smoke parametrization on affected ports.
- `ingress_port_field` gate greps ingress, wire, and settings projection modules for `Protocol`-typed fields — capability must move to constructor or operation parameter at root.
- `interior_isinstance` gate greps non-boundary packages for `isinstance` on port symbols — narrowing belongs at composition root or AOP structural prefix.
- `vendor_in_all` gate inspects public `__all__` exports — port symbol and `resolve_*` factory only; vendor adapter classes indexed negative.
- `orphan_port_symbol` gate compares live `register_law` MANIFEST entries against `PORT_WITNESS_CATALOG` — every subject qualname resolves to catalog row with matching `law_slugs` tuple.
- `checker_parity_gap` gate runs ty, mypy, and pyright on the same representative port matrix module — one green backend does not waive declarations required by others.
- `law_slug_on_path` gate rejects `register_law` calls keyed to pytest module paths instead of `port_symbol` — harness discovery fails at registration smoke before generative suites.
- Gate discovery modules import slug band, `ci_gate_ids`, and `LAW_SLUG_LAYER_MAP` only — they do not import vendor backends or domain interiors; false coupling hides orphan ports without parametrized tests.

# Rejection Shortcuts

- Conformance proof without frozen catalog row — register `port_symbol` in `PORT_WITNESS_CATALOG` before parametrized law modules.
- `register_law("tests.tools.assay...", slug)` keyed to pytest path instead of `port_symbol` — witnesses must bind to port owner for harness discovery.
- Mega-test parametrized on all `law_slugs` for one port on a single axis — one test function per slug per port for failure localization.
- `unittest.mock.Mock` satisfying runtime smoke without structural double from `get_protocol_members` — indexed negative `mock_erased_standin`.
- Generative laws using unfiltered object factories filtered only at runtime — table-driven doubles precede behavior properties.
- Interior domain module importing vendor backend for catalog convenience — bypasses `consumer-negative-import` and export opacity gates.
- Metamorphic proof asserting same adapter instance identity across process spawn — contradicts `adapter_identity_must_differ` on every sub-row crossing process seams.
- Tier traversal property evaluating weaker backends after first `Ok(Certified[C])` — short-circuit supremum is catalog obligation, not optional optimization.
- Catalog row with `ci_gate_ids` tuple but no lint or grep job glob in row metadata — prose-only gate fails done criteria.
- Duplicate `law_slugs` declared in test file parallel to owner registry — collapse to single parametrized catalog import.

# Catalog Completeness Checklist

- Every admitted `port_symbol` has one `WitnessCatalogRow` with `member_frozenset_hash`, closed `law_slugs` tuple, and `evidence_grades` subset matching `LAW_SLUG_LAYER_MAP`.
- Every `law_slugs` entry has `register_law(Port, slug)` witness and exactly one harness layer assignment — discovery smoke passes before generative or metamorphic replay.
- Every `negative_defect_ids` entry has one fixture factory and one assertion family in table-driven negative modules.
- Every `bind_row_id` resolves to composition-root bind record keyed by `type[Port]` — orphan bind or port without factory fails `orphan_port_symbol`.
- Every `ci_gate_ids` entry binds to lint, grep, or matrix job with explicit file globs recorded in row metadata.
- Every `consumer_contexts` entry runs `consumer-negative-import` independently — combined mega-import test discouraged.
- Every refinement chain with `runtime_smoke` in `evidence_grades` declares `refinement_link_ids` with per-link `@runtime_checkable` flags in `runtime_smoke` sub-row.
- Every egress-participating port with `metamorphic_rebind` grade declares metamorphic sub-rows with `handoff_stages` and `adapter_identity_must_differ=true`.
- Every promotion touching port surface emits catalog diff with `pre_hash`, `post_hash`, and `law_slugs` delta.
- Done when catalog registry, bind graph, parametrized harness consumption, CI gate discovery, and rejection shortcuts are closed — interior modules never import witness tables as runtime authority.
