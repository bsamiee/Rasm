# rails-and-effects — core runtime API surface (effect@3.21.2)

# Effect 3.21.2 Runtime API-Truth (RUNTIME lane: rails, effects, concurrency, streams, scheduling, resources, services, observability)

Source: `node_modules/effect/dist/dts/*.d.ts`. Schema/Data/Match/Brand/Machine deliberately excluded.

`[EFFECT_TYPE]:`
- `Effect<out A, out E = never, out R = never>` — A=success, E=typed error channel, R=requirements (contravariant in effect). The R channel is eliminated by provision; E is the rail.
- `Effect extends Pipeable, Inspectable`; STM/Fiber/FiberRef/Latch/Mailbox/RcRef/Pool/ManagedRuntime all `extends Effect` (directly yieldable in `gen`).

`[COMPOSITION_SPINE]:`
- `gen(f: (resume: Adapter) => Generator<YieldWrap<Effect>, AEff, never>) => Effect<AEff, E, R>` — E and R inferred as union of every yielded effect; also a `gen(this, f)` self-bound overload.
- `fn: fn.Gen & fn.NonGen & ((name: string, options?: Tracer.SpanOptions) => fn.Gen & fn.NonGen)` — named/traced function factory: `Effect.fn("span")(function*(...) {...})` adds a span + stack-trace location; trailing pipe args `(effect, ...args) => Effect` form a post-pipeline. `fnUntraced: fn.Untraced` — same without span (perf path).
- `all(arg, options?)` where options = `{ concurrency?: Concurrency; batching?: boolean|"inherit"; discard?: boolean; mode?: "default"|"validate"|"either"; concurrentFinalizers?: boolean }`. arg is `Iterable<Effect>` (→Array result) OR `Record<string,Effect>` (→struct result) OR tuple (→tuple result). `mode:"validate"`→`Array<Option<E>>` error channel (collects all failures); `mode:"either"`→`Array<Either<A,E>>` success channel (never fails); `discard:true`→`void`. `Concurrency = number | "unbounded" | "inherit"`. `allWith(options)` is the data-last pipe form.
- `forEach(iterable, f, options?)` — options `{ concurrency?; batching?; discard?; concurrentFinalizers? }`; `discard:true`→`Effect<void>` else `Array`.
- `filter(iterable, predicate, options?)` effectful predicate `(a) => Effect<boolean>`; `filterMap` via Option-returning effect.
- `when(self, condition)` / `unless(self, condition)` → `Effect<Option<A>,E,R>` (Some on run, None on skip); condition is `LazyArg<boolean>` or effectful predicate.
- `zip(self, that, options?: { concurrent?: boolean })`; `zipWith(self, that, f, options?)`; `zipLeft`/`zipRight`.
- `race(a, b)` — first to complete (success OR failure) wins, loser interrupted; `raceAll(Iterable<Effect>)` — first success; `raceFirst`/`raceWith` lower-level.
- `timeout(self, duration)` → fails `TimeoutException`. `timeoutOption` → `Effect<Option<A>>` (None on timeout, no fail). `timeoutFail(self, { onTimeout: LazyArg<E2>, duration })`. `timeoutFailCause({ onTimeout: LazyArg<Cause>, duration })`. `timeoutTo({ onTimeout, onSuccess, duration })` → fallback value.
- `iterate(initial, { while: (s)=>boolean, body: (s)=>Effect<s> })`; `loop(initial, { while, step, body, discard? })` (collects body results unless discard); `whileLoop({ while: LazyArg<boolean>, body: LazyArg<Effect<A>>, step: (a)=>void }) => Effect<void>` (low-level).
- Do-notation: `Do: Effect<{}>` seed; `bind(name, f: (scope)=>Effect)`; `bindTo(name)`; `let(name, f: (scope)=>value)` (pure binding); `bindAll(f, options?)` (parallel struct bind with `all`-style concurrency options).
- Core constructors: `succeed(A)`, `fail(E)`, `sync(LazyArg<A>)`, `suspend(LazyArg<Effect>)`, `promise((signal)=>PromiseLike<A>)`, `tryPromise({ try, catch })`, `async((resume, signal)=>void|Effect, blockingOn?)`. `map`/`flatMap`/`andThen`/`tap`/`as`/`asVoid`. `andThen` accepts Effect | (a)=>Effect | Promise | value (polymorphic sequencer).

`[ERROR_RAIL]:`
- `catchAll(self, (e)=>Effect)` — handles E, leaves Cause defects. `catchAllCause(self, (cause)=>Effect)` — handles full Cause (incl. Die/Interrupt). `catchAllDefect((defect)=>Effect)` — only unrecoverable defects.
- `catchTag(self, tag, (e)=>Effect)` and `catchTags(self, { TagA: handler, TagB: handler })` — discriminate on `_tag` field; narrows residual E. `catchIf(self, refinement|predicate, handler)`. `catchSome(self, (e)=>Option<Effect>)`; `catchSomeCause`; `catchSomeDefect`.
- `mapError(self, (e)=>e2)`; `mapBoth(self, { onFailure, onSuccess })`.
- `orElse(self, ()=>that)`; `orElseFail(self, ()=>e2)`; `orElseSucceed(self, ()=>a)`; `orDie(self)` (E→defect, E=never); `orDieWith(self, (e)=>defect)`.
- `retry(self, policyOrOptions)` — policy is `Schedule<X, E, R>` OR options `{ while?: (e)=>boolean|Effect<boolean>; until?: (e)=>boolean|Effect<boolean>; times?: number; schedule?: Schedule<any,E,any> }` (Retry.Options). `retryOrElse(self, policy, (e, out)=>fallback)`.
- `Cause<E> = Empty | Fail<E> | Die | Interrupt | Sequential<E> | Parallel<E>` — Fail=typed error, Die=defect(unknown), Interrupt=fiberId; Sequential/Parallel compose causes (parallel = concurrent failures). Constructors `Cause.fail/die/interrupt/empty/sequential/parallel`. Guards `isFailType/isDie/isInterrupt/isEmpty/isFailure`. Projections `failures(self): Chunk<E>`, `defects(self): Chunk<unknown>`, `keepDefects`, `stripFailures`, `squash`/`squashWith`, `pretty(cause, options?)`, `prettyErrors`.
- `sandbox(self): Effect<A, Cause<E>, R>` (expose full cause as error); `unsandbox` inverse. `cause(self): Effect<Cause<E>, never, R>`.
- `flip(self): Effect<E, A, R>` (swap channels); `flipWith`. `merge(self): Effect<A|E, never, R>` (collapse both to success).
- Projections: `either(self): Effect<Either<A,E>, never, R>`; `option(self): Effect<Option<A>, never, R>`; `exit(self): Effect<Exit<A,E>, never, R>`. `ignore`/`ignoreLogged`.
- `Exit<A,E> = Success<A> | Failure<Cause<E>>`; `Exit.succeed/fail/failCause/die/interrupt`, `isSuccess/isFailure`, `map/mapError/flatMap/match/matchEffect/getOrElse`.
- Taps (rail-preserving): `tapError(self, (e)=>Effect)`; `tapErrorTag(self, tag, handler)`; `tapErrorCause(self, (cause)=>Effect)`; `tapDefect(self, (defect)=>Effect)`; `tapBoth(self, { onFailure, onSuccess })`.
- Rail annotations: `annotateLogs(self, key, value)` / `annotateLogs(self, {...})`; `annotateSpans(self, key, value)`.

`[DEPENDENCY_SURFACE]:`
- `Context.Tag<Id, Value>` — `Context.Tag(id)<Self, Shape>()` class-form, or `Context.GenericTag<Id, Service>(key)`. `Context` ops: `make(tag, service)`, `add`, `get`, `getOption`, `merge`, `mergeAll`, `pick(...tags)`, `omit(...tags)`, `empty()`.
- `Context.Reference<Self>()(id, { defaultValue: () => Service })` → `ReferenceClass` — a Tag with a built-in default, so it resolves WITHOUT being provided (defaults the runtime value). The mechanism for ambient defaults (Clock, Random, ConfigProvider use this).
- `Effect.Service<Self>()(key, make)` — `make` is `{ effect | scoped | sync | succeed; dependencies?: ReadonlyArray<Layer>; accessors?: boolean }`. Produces a class that is simultaneously the `Context.Tag<Self,Self>`, has `.Default` Layer (or `.DefaultWithoutDependencies` + dependency-merged `.Default` when `dependencies` present), `.make(impl)`, `.use(body)`, and (when `accessors:true`) per-method accessor proxies. `scoped` make → Default Layer carries `Scope` lifetime.
- `Effect.provide(self, layerOrContext|Array<Layer>)`; `provideService(self, tag, impl)`; `provideServiceEffect(self, tag, effect)`.
- Layer algebra: `Layer<ROut, E, RIn>`. Constructors: `succeed(tag, impl)` / `sync(tag, lazy)` / `effect(tag, effect)` / `scoped(tag, scopedEffect)` / `scopedDiscard` / `context<R>()` / `service(tag)` / `effectContext` / `fail` / `die` / `empty`.
- Wiring: `provide(self, that)` (that's RIn satisfied by self, self's ROut hidden), `provideMerge(self, that)` (same but ROut exposed in output), `merge(a, b)`, `mergeAll(...layers)`. `discard(self)` drops output.
- Memoization: layers are SHARED by default — the same layer instance built twice in a graph is constructed once (memoized per build). `fresh(self)` forces a non-shared rebuild. `memoize(self): Effect<Layer, never, Scope>` returns a pre-memoized handle. `MemoMap` is the explicit memoization registry; `buildWithMemoMap` threads it.
- Build/launch: `build(self): Effect<Context<ROut>, E, Scope|RIn>`; `buildWithScope(self, scope)`; `launch(self): Effect<never, E, RIn>` (builds + holds forever, for long-running app layers); `toRuntime(self): Effect<Runtime<ROut>, E, Scope>`.
- Rail on layers: `orDie`, `orElse`, `catchAll`, `retry(self, schedule)`, `map`, `mapError`, `tap`. `setClock`/`setConfigProvider`/`setTracer` produce ambient-override layers.

`[RESOURCE_LAW]:`
- `Scope` extends Pipeable; `CloseableScope extends Scope`. `Scope.make(executionStrategy?): Effect<CloseableScope>`; `Scope.close(scope, exit)`; `Scope.addFinalizer(scope, effect)` / `addFinalizerExit(scope, (exit)=>effect)`; `Scope.fork(scope, strategy)` (child scope); `Scope.extend`/`Scope.use`. `Scope.Scope` is the Tag.
- `acquireRelease(acquire, (resource, exit)=>release)` → `Effect<A, E, Scope|R>`; acquire AND release run uninterruptibly; release added to Scope finalizers, runs on scope close with terminal Exit. `acquireReleaseInterruptible` (interruptible acquire). `acquireUseRelease(acquire, use, release)` — bracket without requiring Scope in R.
- `addFinalizer((exit)=>Effect): Effect<void, never, Scope|R>` — registers a scope finalizer inline.
- `ensuring(self, finalizer)` (always runs, ignores exit); `onExit(self, (exit)=>Effect)`; `onError(self, (cause)=>Effect)`; `onInterrupt(self, (interruptors)=>Effect)`; `ensuringChild`.
- `scoped(self): Effect<A, E, Exclude<R, Scope>>` — provides a fresh Scope, closes it on completion (eliminates Scope from R). `scopedWith((scope)=>Effect)` — explicit scope handle.
- `using(self, (resource)=>use)` — scoped-resource use form. `parallelFinalizers`/`sequentialFinalizers`/`finalizersMask(strategy)` control finalizer execution strategy.
- FiberRef: `FiberRef<A> extends Effect<A>` (yielding reads). `make(initial, { fork?: (a)=>a, join?: (left,right)=>a }): Effect<FiberRef<A>, never, Scope>` — `fork` transforms value when a child fiber forks, `join` merges back on fiber join. `unsafeMake` (no Scope). Ops `get`/`set`/`update`/`modify`/`getAndSet`/`reset`. `Effect.locally(self, fiberRef, value)` / `locallyWith(self, fiberRef, (a)=>a)` — scoped override for the duration of self (the canonical reader-context override).

`[CONCURRENCY_FORK]:`
- `fork(self): Effect<RuntimeFiber<A,E>, never, R>` — child fiber, interrupted when parent scope/fiber ends (structured). `forkDaemon` — top-level fiber, NOT bound to parent (survives parent). `forkScoped: Effect<RuntimeFiber, never, Scope|R>` — bound to enclosing Scope. `forkIn(self, scope)` — bound to a specific Scope. `forkAll(effects, options?)`. `forkWithErrorHandler(self, handler)`.
- Fiber: `Fiber<A,E> extends Effect<A,E>` (yielding = join). `RuntimeFiber<A,E> extends Fiber`. `await(self): Effect<Exit<A,E>>`; `join(self): Effect<A,E>` (propagates failure); `interrupt(self): Effect<Exit<A,E>>` (awaits interruption); `interruptFork` (fire-and-forget interrupt); `poll(self): Effect<Option<Exit<A,E>>>`; `inheritAll`; `children`; `roots`. Collection forms: `awaitAll`, `joinAll`, `interruptAll`, `Fiber.all`, `Fiber.zip`, `Fiber.orElse`, `Fiber.map`.
- Interruption model: `interrupt: Effect<never>` (self-interrupt); `uninterruptible(self)` / `interruptible(self)`; `uninterruptibleMask((restore)=>...)` / `interruptibleMask((restore)=>...)` — mask region, `restore` re-opens interruptibility for sub-effects. `disconnect(self)` (detaches interruption to background). `withConcurrency(self, n)`.
- `Effect.makeSemaphore(permits): Effect<Semaphore>` — canonical concurrency limiter. `Semaphore.withPermits(n)(effect)` (acquire N, run, release), `withPermitsIfAvailable(n)` → `Option<A>` (no-run if unavailable), `take(n)`/`release(n)`/`resize(n)`. `unsafeMakeSemaphore`.
- `Effect.makeLatch(open?): Effect<Latch>` — `Latch extends Effect<void>` (yield = await). `.open`/`.close`/`.release`/`.await`/`.whenOpen(effect)`.
- Deferred: one-shot single-assignment. `make<A,E>(): Effect<Deferred<A,E>>`; setters each `Effect<boolean>` (true=won): `succeed/fail/failCause/die/complete/completeWith/done(exit)/interrupt`. `await(self): Effect<A,E>`; `poll(self): Effect<Option<Effect<A,E>>>`; `isDone`.
- Queue: `bounded/unbounded/dropping/sliding(capacity): Effect<Queue<A>>`. bounded=`offer` backpressures producer; dropping=`offer` returns false + discards new when full; sliding=`offer` always succeeds + evicts oldest. `take: Effect<A>`, `takeAll: Effect<Chunk<A>>`, `takeUpTo`/`takeBetween(min,max)`/`takeN(n)`, `poll: Effect<Option<A>>`, `offer`/`offerAll`, `size`, `capacity` (plain), `shutdown`/`awaitShutdown`/`isShutdown`. `Dequeue<A>`/`Enqueue<A>` split.
- PubSub: `bounded/dropping/sliding(number | {capacity, replay?})`, `unbounded(options?)`. `publish`/`publishAll`. `subscribe(self): Effect<Dequeue<A>, never, Scope>` — each subscriber gets own Dequeue, auto-unsubscribed on scope close. Same backpressure model as Queue.
- Ref: `make(A): Effect<Ref<A>>`; `get/set/update/updateAndGet/getAndUpdate/getAndSet/setAndGet/modify/modifySome/updateSome`; `unsafeMake`. Atomic for PURE functions only. SynchronizedRef: adds mutex-serialized effectful updates `updateEffect/updateAndGetEffect/getAndUpdateEffect/modifyEffect/modifySomeEffect`. SubscriptionRef: `.changes: Stream<A>` field — emits current then every change (Ref + observable).
- Fiber-lifecycle collections (all `make: Effect<X, never, Scope>`, scope-close interrupts owned fibers): FiberHandle (single, `run`/`set`/`get`/`clear`/`join`); FiberMap (keyed dedup, `run`/`set`/`get`/`has`/`remove`/`size`/`join`/`awaitEmpty`); FiberSet (dynamic unkeyed, `run`/`add`/`clear`/`size`/`join`/`awaitEmpty`).
- Mailbox: `make(capacity? | {capacity, strategy?: "suspend"|"dropping"|"sliding"}): Effect<Mailbox<A,E>>` — bounded async queue WITH completion signal. Write: `offer`/`offerAll`/`fail`/`failCause`/`done(exit)`/`end`/`shutdown`. `ReadonlyMailbox extends Effect<[Chunk<A>, done: boolean], E>` (yield drains a batch + completion bit); `takeAll`/`takeN`/`take`/`await`/`clear`/`size`. `Mailbox` (rw) extends `ReadonlyMailbox`.
- RcRef/RcMap: reference-counted shared scoped resources with lazy TTL teardown. `RcRef.make({acquire, idleTimeToLive?})`; `RcRef extends Effect<A,E,Scope>`, `get` increments refcount + releases on caller scope close, teardown after refcount=0 AND TTL; `invalidate`. `RcMap.make({lookup: (k)=>Effect, idleTimeToLive?: Duration|(k)=>Duration, capacity?})`; `get(key)`/`keys`/`invalidate(key)`/`touch(key)`. Lifetime lives in the `R=Scope` channel — scope granularity is the real teardown knob, not TTL.
- STM: `STM<A,E,R> extends Effect` (yields commit implicitly in `Effect.gen`, stays transactional in `STM.gen`). `commit(self): Effect<A,E,R>` (atomic; auto-retry on read-set conflict — optimistic concurrency). `STM.retry: STM<never>` (block until a read TRef changes), `check(LazyArg<boolean>)`, `cond(pred, error, result)`, `succeed/fail`, `catchAll/catchSome/orElse/retryWhile/retryUntil`. Transactional refs (all ops return STM, compose then commit once): TRef, TMap, TArray, TQueue (bounded/unbounded/dropping/sliding), TSet, TSemaphore (`withPermit(s)`/`withPermitsScoped`), TDeferred (`poll: STM<Option<Either<A,E>>>`), TPubSub (`subscribe: STM<TDequeue>` / `subscribeScoped: Effect<TDequeue, never, Scope>`), TPriorityQueue (needs `Order<A>`).

`[STREAM]:`

- Transforms: `mapAccum(s, (s,a)=>[s2,a2])` (+Effect); `scan`/`scanEffect`/`scanReduce`; `grouped(n)`; `groupedWithin(n, duration)`; `throttle({cost: (chunk)=>number, units, duration, burst?, strategy?: "enforce"|"shape"})` (default "shape"; token-bucket — enforce drops, shape delays); `debounce(duration)`; `buffer({capacity:"unbounded"} | {capacity, strategy?: "dropping"|"sliding"|"suspend"})`, `bufferChunks`/`bufferDropping`/`bufferSliding`; `aggregate(sink)`, `aggregateWithin(sink, schedule)` (Right-only), `aggregateWithinEither` (Sink leftover re-fed — leftover type L is load-bearing); `mapEffect(f, {concurrency?, unordered?})` (+keyed `{key, bufferSize?}` overload); `flatMap(f, {concurrency?, bufferSize?, switch?})` (switch:true = latest-wins/switchMap); `merge(that, {haltStrategy?: HaltStrategy|"left"|"right"|"both"|"either"})`, `mergeAll({concurrency, bufferSize?})`, `mergeWithTag`/`mergeLeft`/`mergeRight`/`mergeEither`; `zip`/`zipLatest`/`zipLatestAll`/`zipWith`/`zipWithLatest`; `broadcast(n, lag)`/`broadcastDynamic`/`broadcastedQueues` (Effect<…, Scope>); `share({capacity, strategy?, replay?, idleTimeToLive?})` (idleTimeToLive share-only auto-teardown); `partition(pred, {bufferSize?})`/`partitionEither`; `retry(schedule)`, `schedule(schedule)`, `repeat(schedule)`; `tap`/`tapError`/`tapSink`; `mapChunks`/`filterMapEffect`/`rechunk(n)`/`changes`; `interruptWhen(effect)`/`haltWhen(effect)`; `timeout`/`timeoutFail`/`timeoutTo`.
- Sink: `Sink<out A, in In=unknown, out L=never, out E=never, out R=never>` — A=result, L=leftover. `fold(s, contFn, f)`/`foldLeft`/`foldUntil`/`foldWeighted({initial, maxCost, cost, body})`/`foldWeightedDecompose`; `collectAll`/`collectAllN(n)`/`collectAllToMap(key, merge)`/`collectAllWhile`; `forEach`/`forEachChunk`/`forEachWhile` (leftover on stop); `head`/`last` (Option, leftover), `count`/`sum`/`drain` (values); `zip(that, {concurrent?})`/`zipLeft`/`zipRight`/`zipWith` (concurrent:true = parallel sinks same input); `race`/`raceBoth`; `map`/`mapInput`/`mapInputChunks`/`mapInputEffect`/`dimap({onInput,onDone})`/`flatMap`/`mapEffect`; `ignoreLeftover`/`collectLeftover` (→`[A, Chunk<L>]`).

`[SCHEDULE]:`
- `Schedule<out Out, in In = unknown, out R = never>` — Out=accumulated decision/output (covariant), In=consumed input (contravariant: E under retry, A under repeat), R=env. SAME schedule drives both `Effect.retry` (In=E) and `Effect.repeat` (In=A) precisely because In is contravariant. `makeWithState(initial, step)`.
- Primitives: `exponential(base, factor?=2)`, `spaced(d)`, `fixed(interval)`, `linear(base)`, `recurs(n)`, `once`, `forever`, `stop`, `identity<A>()`, `fromDelay`/`fromDelays`, `fromFunction`, `succeed`, `duration(d)`, `windowed(interval)`, `secondOfMinute/minuteOfHour/hourOfDay/dayOfWeek/dayOfMonth(n)` (calendar-aligned), `cron(Cron.Cron)` OR `cron(expression: string, tz?)` → `Schedule<[number,number]>`, `unfold`, `count`, `elapsed`, `fibonacci(one)`. NO `Schedule.fail`.
- Combinators: `union`/`unionWith` (either; shorter delay), `intersect`/`intersectWith` (both; longer delay), `andThen`/`andThenEither`, `compose`, `jittered` (0.8–1.2), `jitteredWith({min?, max?})`, `whileInput`/`whileOutput`/`untilInput`/`untilOutput` (+Effect), `recurWhile`/`recurUntil` (+Effect), `check`/`checkEffect`, `tapInput`/`tapOutput`, `modifyDelay`/`modifyDelayEffect`, `delayed`/`delayedEffect`/`delayedSchedule` (Duration-output self-feed), `addDelay`/`addDelayEffect`, `resetAfter`/`resetWhen`, `collectAllInputs`/`collectAllOutputs`/`collectWhile`/`collectUntil`, `map`/`mapEffect`/`as`, `mapInput`/`mapInputEffect`/`mapInputContext`, `zipLeft`/`zipRight`, `repetitions`, `reduce`/`reduceEffect`, `passthrough` (output=input), `ensuring`, `bothInOut`, `either`/`eitherInOut`. NO `both`/`first`/`second`/`dimap` as exports.
- Driver: `driver(self): Effect<ScheduleDriver<Out,In,R>>`. `ScheduleDriver`: `next(input: In): Effect<Out, Option<never>, R>` (success Out=continue one step; fail with `Option.None`=schedule done), `last: Effect<Out, NoSuchElementException>`, `reset`, `state`.
- Cron module: `Cron { tz: Option<TimeZone>; seconds/minutes/hours/days/months/weekdays: ReadonlySet<number> }`. `make({minutes, hours, days, months, weekdays /*required*/, seconds?, tz?})` (Iterable<number> in). `parse(string, tz?): Either<Cron, ParseError>` (`_tag:"CronParseError"`; accepts 6-field with leading seconds). `unsafeParse`, `match(cron, date)`, `next(cron, startFrom?: DateTime.Input): Date`, `sequence(cron, startFrom?): IterableIterator<Date>`, `equals`, `isCron`.

## Further Considerations

- The `mode` axis on `Effect.all`/`forEach` is the typed-error-accumulation lever that most consumers miss: `mode:"validate"` moves ALL failures into an `Array<Option<E>>` error channel (parallel error collection, the Validation-applicative analog) while `mode:"either"` makes the combinator total (`Array<Either<A,E>>`, never fails). Reaching for manual `Effect.either` + partition is reinventing this. For STM, the same accumulation is impossible — STM has no validate mode, so cross-TRef multi-error gathering must commit-then-aggregate at the Effect layer.
- `ScheduleDriver.next` encoding (success `Out`=continue, `fail(Option.None)`=exhausted) and `Mailbox`/`ReadonlyMailbox` yielding `[Chunk<A>, done: boolean]` are two places where the control signal rides a channel that looks like data — a C#-side projection or any manual driver MUST thread the discriminant (the `done` bit, the `None`) rather than the payload, or it silently loses termination. Pair `driver.next` with `driver.last` (fails `NoSuchElementException`) to recover the final emitted value.
- Lifetime correctness in this runtime is almost entirely encoded in the `R = Scope` channel, not in explicit close calls: `forkScoped`, `PubSub.subscribe`, `TPubSub.subscribeScoped`, `RcRef.get`, `Pool.get`, `acquireRelease`, scoped `Effect.Service` all defer teardown to the enclosing scope. The real tuning knob is scope GRANULARITY — an `RcRef.get` or `forkScoped` whose enclosing scope outlives intended use pins the resource/fiber and defeats refcount TTL or structured interruption. `Effect.scoped` is the boundary that converts a `Scope`-requiring effect into a self-contained one; place it at the narrowest correct frame.
