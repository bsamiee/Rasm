# [HARVEST]

Per-round dispatch for the distill leg. Claude rides `Agent(subagent_type: "reviewer-harvest")` with the filled `[01]` round slots as the whole prompt — the agent profile carries the law, the model chosen at dispatch. Codex takes the `[02]` law block as developer instructions and the filled round slots as the user prompt. `[02]` carries `.claude/agents/reviewer-harvest.md` in codex form — one law, two spellings: an edit to either surface lands on both in the same pass. Fill every slot; a slot the round genuinely lacks is omitted whole, never left as a placeholder.

```markdown template
User directives: <round-scoped instructions>.
```

```markdown template
<decision_procedure>
1. DEDUP: co-arriving same-root lessons cluster into one candidate before classification, every class carrying a stable id through the ledger. A lesson and a ruling share a root when they govern the same defect-or-hunt family on the same owner; a lesson spanning several sharing rulings absorbs them all into one owner.
2. PROVENANCE routes disposition: relitigations of dispositioned claims are harden-or-skip material carrying priority; new-work findings on files the prior wave edited grade the wave and feed steering; untouched-file findings mark reviewer-depth change — ledger-recorded, never guard fodder. Rising new-work share beside falling relitigation is convergence.
3. PRECEDENCE: the feed governs scope and emphasis; the standing invariants — lesson-never-instance, integration-over-accretion, altitude, mirror, grain — are non-overridable, and a feed directive violating one lands as a defects row, unexecuted.
4. Each (class, surface) pair resolves to skip, harden, or new owner:
   - SKIP only when the owning ruling already carries the class's specific payload — a refuted class against an already-sharpened guard, re-review of one uncommitted diff re-minting findings regardless of wording; a steering class only when its exemplar rides the owning hunt guard; axis-level ownership never earns it.
   - HARDEN widens in place the guard whose wording was too narrow to bite, never a duplicate row; recurred classes outrank new material, and [GUARD_INEFFECTIVE] marks failed wording by definition — harden its owner, never skip it as owned.
   - NEW OWNER only when no ruling on that surface shares the root — an id-bearing Greptile sibling, a fresh clause in the owning CodeRabbit path block, or a new one-topic Macroscope file.
5. INTEGRATION OVER ACCRETION: a lesson adding coverage its owning ruling lacks merges into one ruling covering both better than either — the owning clause rewritten so the fact reads integral, never appended. A merged owner takes its absorbed rows' maximum severity and scope-glob union; a mis-targeting union proves the roots distinct. A file opened for any landing leaves whole-file hardened: every near-similar pre-existing pair — not only rows sharing the incoming root — merges into one higher-resolution rule keeping both decisions, replaced rows deleting with protected payload intact, every line whose removal loses no flag-or-no-flag decision deletes, and a merge grown in words without gaining a decision is redone; harden rides touch, never reach.
6. LESSON, NEVER INSTANCE: every fix or refutation abstracts to the universal pattern — the owner shape, the admission fold, the collapse — never corpus symbols, file names, page names, lane ids, or round numbers, on every surface, caption, and fence comment. Two carve-outs, both external members: a decompile-verified host spelling rides its guard verbatim as evidence roster; a verified external member proving a stacking or collapse gap rides the owning hunt guard as a leverage exemplar.
7. ALTITUDE: a lesson is universal only when it bites with the language, type system, and library stripped away — family completion, admission folds, preimage completeness, shared-owner guards, structural unspellability, mint-time evidence binding, intake canonicalization illustrate the test; everything else is language-bound, landing only in language-scoped rows, blocks, or files. CodeRabbit's global home is the path: "**" instructions block.
8. BOTH FRAMINGS, ONE STANCE: negative framing kills false classes — do-not-flag guards citing the refuting ruling; positive framing steers to stronger demands — hunt axes (API-stacking, hand-rolling, strata non-leverage, wrong-altitude homing, under-parameterization, under-scoped owners, shape collapse, dual-paradigm seams, hand-mirrored vocabularies, entry-point quality, single-use-helper inlining), unconsumed verified capability as a gap, and strongest-form presumption: conformant-but-weak is unfinished. Neither alone: suppression starves the stream, steering re-burns refuted classes.
9. MIRROR + GRAIN: a fact lands on each surface whose model owns its altitude and scope, one owner per fact per surface; a surface with no owning mechanism records a skip naming why — never a forged counterpart, never a deferred one. Refuted classes and proven hit-shapes land on every engine surface whichever engine raised them — an unguarded engine re-mints a refuted class, and a proven shape is worth hunting from every engine that sees it; evidence rides the raising surface as the refuting ruling or the landed form, siblings carrying the same law in their own grain, never a copied sentence.
```
