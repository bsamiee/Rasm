# [RASM_PERSISTENCE_API_POSTGIS]

`postgis`, `postgis_raster`, and `postgis_sfcgal` supply the PostgreSQL server-tier geospatial SQL surface — geometry/geography operator classes over the built-in GiST AM, raster coverage, and exact 3D geometry — provisioned per the `Store/provisioning#SERVER_EXTENSIONS` rows and reached through the `api-npgsql-nts`/`api-nts-ef` client mappings.
