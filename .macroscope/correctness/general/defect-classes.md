Proven defect classes hunted at root-cause depth on every surface; each paragraph is one hunting family.

Standing classes: required-field threading dropped by signature migrations, finiteness admission before range checks — family-wide across the policy roster, zero-sentinel law stated beside the guard — identity preimages omitting output-affecting inputs, match/dispatch totality, frozen-payload discipline — mutable defaults, unsorted iteration reaching egress — per-stream timestamp re-basing, resource custody on every exit path, archive and metadata bomb gates with intake bound to admitted identity — retained handles, handle-relative no-follow traversal, handle-derived size accounting, never re-resolved path strings — deterministic egress ordering, and identity, rank, or grouping derived from traversal or enumerator order rather than a domain-canonical key — implementation-owned nondeterminism whose fix is the canonical order or the domain-stable identity minimum.

Identity and keys: a multi-field hash preimage length-frames every variable-width field and count-frames every adjacent collection; separator-joined concatenation is rejected outright. A derived artifact keys on the content hash of its source, never path or mtime. Root discovery walks upward to a sentinel file, never a fixed parent depth. A wire token admits only its exact emission spelling; a tolerant parse re-emitting a normalized form forks the key. Execution policy and display labels stay outside a content preimage — a timeout or retry budget changes when work fails, never the bytes a success produces, and a label inside the preimage is a collision defect; a security identity takes a cryptographic digest, never a speed hash, and a composite id rides a canonical codec, never a hand-rolled join or quote scheme injective on one ambiguity axis. A cache splits identity slot from content address — the identity slot carries the pre-run source key the hit test compares, the produced output's content address is a separate derived fact, and minting the output address into the identity slot silently defeats keyed elision; a source with no canonical byte form (live handle, callable, non-deterministic serialization) joins the key as environment-scoped identity and demotes admission to forced-live. A parent-ancestry or dependency-multiset key folds as an ordered incremental hash over sorted parent keys — an XOR or additive fold self-cancels duplicate identical-content parents.
