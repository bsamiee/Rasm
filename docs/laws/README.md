# [LAWS_CORPUS]

Repo-wide maintenance law: the coupling topology every cross-surface edit obeys, the pattern rows binding more than one language branch, and the scar ledger of regression-proven laws. The heavy workflows (`rebuild`, `finalize`, `estate`) land their generalizable findings here through one terminal doctrine stage; this corpus is the system of record only for facts no other owner carries.

A landing hardens an existing row before extending a page, and extends a page before minting one; a new page requires a law family no existing page charters. A fact owned elsewhere never gains a copy here — the owner keeps it, and a topology row may name it as a counterpart. A row earns admission by recurrence across independent units or by a structural singleton whose blast radius spans branches; the landing stage records that evidence in its run report, and the row itself states only the law.
