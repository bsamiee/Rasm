# [PY_ARTIFACTS_PLAN]

The content-keyed artifact-production planning layer over the runtime session lane. `ArtifactPipeline` is ONE plan owner that folds a producer dependency graph into the `runtime/execution#LANE` admission shape: each `ArtifactWork` node carries a producer's pre-minted `ContentIdentity`-derived `ContentKey`, its `Work[ArtifactReceipt]` coroutine, and the closed `Admission` case it enters the lane on, the `graphlib.TopologicalSorter` resolves the dependency-level fronts, and each node lowers through its own `ArtifactWork.lowered` admission `match` into its `runtime/execution#LANE` `Admit` case — a `keyed` node to the `(ContentKey, Work)` cache unit so a downstream front re-admitting an already-produced artifact replays the upstream `Ok` from the threaded session cache rather than re-rendering, a `bare` node to a forced-live unit, a `retried` node to a transient-guarded unit re-spawning a worker cold-start under the `RetryClass` its admission case mandates. It owns NO cache, NO store, NO scheduler, NO drain, and NO retry binding — the `runtime/execution#LANE` `LanePolicy` owns the `CapacityLimiter`/`move_on_after`/`DrainReceipt`, the front-to-front `DrainReceipt.cache` thread, and the concurrent per-front drive, and the `runtime/execution#LANE` `Admit` cases own the content short-circuit, the always-run, and the `guard(cls)` retry; `ArtifactPipeline` only PROJECTS the producer graph into those runtime ports, resolving the front ORDER while the lane resolves the front DRIVE, and surfaces its own topology and DAG-shape evidence — `keys`/`dangling`/`elided`/`cyclic` the coverage sets, `profile`/`cardinality`/`depth`/`width`/`contention`/`live` the derived shape (`contention` the peak over-subscription of the lane's `CapacityLimiter` the front structure conceals), `roots`/`leaves`/`critical_path`/`span` the source-artifact, terminal-deliverable, cost-weighted-serial-latency, and longest-hop-chain DAG facts over the `requires` adjacency and the per-node `cost` weights, and `severed: Option[PlanFault]` the cause-carrying coverage verdict (`dangling` versus `cyclic`, never a `bool` erasing which). The runtime `StagePlan` is NOT this driver: it keys stages by `str` and forces every unit `retried`, so it cannot host a `ContentKey`-keyed graph whose nodes mix `keyed`/`bare`/`retried` admission; the plan emits the front-ordered `Block[Admit]` tuple the lane's `drain` folds over directly, never a `StagePlan` re-spelling. The plan is the third reuse-fabric consumer of the one `core/receipt#RECEIPT` `contribute` fold beside the `runtime/execution` lane elision and the `runtime/observability` `MeterProvider` signal stream — it reads each producer's contributed `ArtifactReceipt` as the content-keyed evidence the elision distinguishes hit from miss on, AND emits its own `planned`-phase `Receipt` `fact` (the `core/receipt#SIGNALS [PLAN_FABRIC]` consumer obligation) onto that one stream so the planning stage is observable beside the `admitted`/`emitted` producer facts. `ArtifactPipeline.plan` returns a `RuntimeRail[PipelinePlan]` carrying the front-ordered `Block[Admit[ArtifactReceipt]]` lanes keyed by the runtime content key — emitting the planned-phase fact on the resolved-plan arm — never an executed result.

```python signature
class ArtifactWork(Struct, frozen=True):
    key: ContentKey
    work: Work[ArtifactReceipt]
    parents: tuple[ContentKey, ...] = ()
    admission: Admission = Admission(keyed=None)
    # the producer's own estimated work weight (render-time seconds, byte-volume, or a unit default)
    # the WEIGHTED critical path sums along the longest chain; an `elided` cache hit contributes none.
    cost: float = 1.0

    def lowered(self, /) -> Admit[ArtifactReceipt]:
        match self.admission:
            case Admission(tag="keyed"):
                return Admit(keyed=(self.key, self.work))
            case Admission(tag="bare"):
                return Admit(bare=self.work)
            case Admission(tag="retried", retried=cls):
                return Admit(retried=(cls, self.work))
            case _ as unreachable:
                assert_never(unreachable)


class PipelinePlan(Struct, frozen=True):
    # `fronts` is the drive input (one `LanePolicy.drain(front, cache)` per element); `requires` is the
    # producer-graph adjacency the DAG-shape evidence (`roots`/`leaves`/`critical_path`) derives from; every
    # other field is the topology evidence the runtime does NOT own and `severed` gates before the drive.
    fronts: tuple[Block[Admit[ArtifactReceipt]], ...]
    requires: Map[ContentKey, tuple[ContentKey, ...]] = Map.empty()
    weights: Map[ContentKey, float] = Map.empty()
    lane: LanePolicy = _DEFAULT_LANE
    cache_seed: Map[ContentKey, ArtifactReceipt] = Map.empty()
    keys: frozenset[ContentKey] = frozenset()
    dangling: frozenset[ContentKey] = frozenset()
    elided: frozenset[ContentKey] = frozenset()
    cyclic: frozenset[ContentKey] = frozenset()

    @property
    def profile(self) -> tuple[int, ...]:
        return tuple(len(front) for front in self.fronts)

    @property
    def cardinality(self) -> int:
        return sum(self.profile)

    @property
    def depth(self) -> int:
        return len(self.fronts)

    @property
    def width(self) -> int:
        return max(self.profile, default=0)

    @property
    def contention(self) -> int:
        # the peak over-subscription the lane's one `CapacityLimiter` will QUEUE rather than run: a plan
        # whose widest front exceeds `lane.capacity` has a hidden serialization the front structure conceals,
        # so the composition root reads `max(0, width - capacity)` (zero for a plan the lane absorbs) before
        # the drive instead of recombining `width` and `lane` by hand — the saturation verdict the front-count
        # `depth` and the `width` peak alone do not answer.
        return max(self.width - self.lane.capacity, 0)

    @property
    def live(self) -> int:
        # units that reach a coroutine: total minus the `elided` cache hits, the saving the plan exists for.
        return self.cardinality - len(self.elided)

    @property
    def adjacency(self) -> Map[ContentKey, tuple[ContentKey, ...]]:
        # the in-set predecessor projection `roots`/`leaves`/`critical_path` each read — every node's
        # `requires` parents narrowed to the keys the set actually mints, dropping the `dangling` out-of-set
        # edge once so the DAG-shape folds never re-filter it. Recomputed per access from `requires ∩ keys`
        # (a frozen `Struct` carries no writable slot for a `cached_property`); derived, never stored beside
        # `requires` it is recoverable from.
        return self.requires.map(lambda _key, parents: tuple(parent for parent in parents if parent in self.keys))

    @property
    def roots(self) -> frozenset[ContentKey]:
        # the source artifacts — minted nodes with no in-set parent, the terminal-upstream producers a
        # warm replay seeds first; a node whose every parent is `dangling` (out-of-set) is still a root.
        return frozenset(key for key in self.keys if not self.adjacency.try_find(key).default_value(()))

    @property
    def leaves(self) -> frozenset[ContentKey]:
        # the terminal deliverables — minted nodes no other in-set node depends on (a report binding
        # figures is a leaf, the figures it binds are not), the outputs the composition root surfaces.
        return self.keys - frozenset(parent for parents in self.adjacency.values() for parent in parents)

    def _heights(self) -> Map[ContentKey, tuple[float, int]]:
        # the ONE parents-first longest-path memo both chain metrics derive from — a single `Block.fold` over
        # the `graphlib.static_order` sequence accumulating each node's `(cost-weight, hop)` height pair, where
        # an `elided` cache hit contributes zero cost (it never runs) but still one hop. `h(k) = (cost(k) +
        # max(parent costs), 1 + max(parent hops))` reads every in-set parent already settled, so neither
        # projection re-walks the graph and a diamond's shared spine settles once, never the per-root open
        # recursion that grows the stack on a deep chain and recomputes the spine exponentially.
        adjacency, elided = self.adjacency, self.elided
        weight = lambda key: 0.0 if key in elided else self.weights.try_find(key).default_value(1.0)
        order = TopologicalSorter[ContentKey](dict(adjacency.items()))

        def settle(memo: Map[ContentKey, tuple[float, int]], key: ContentKey) -> Map[ContentKey, tuple[float, int]]:
            parents = adjacency[key]
            cost = weight(key) + max((memo[parent][0] for parent in parents), default=0.0)
            hop = 1 + max((memo[parent][1] for parent in parents), default=0)
            return memo.add(key, (cost, hop))

        return Block.of_seq(order.static_order()).fold(settle, Map.empty())

    @property
    def critical_path(self) -> float:
        # the COST-weighted longest dependency chain — the true serial-latency lower bound the lane's
        # concurrency cannot collapse, distinct from `depth` the front count AND from `span` the hop count: a
        # wide-but-shallow plan parallelizes, a deep OR an expensive chain does not, and a 400ms `scene` render
        # dominates a chain of five 5ms tables a bare hop count would rank longer.
        return max((height[0] for height in self._heights().values()), default=0.0)

    @property
    def span(self) -> int:
        # the longest dependency chain by HOP COUNT, the topological depth of the deepest path — the
        # structural counterpart to the cost-weighted `critical_path`, distinct from `depth` the FRONT count
        # (which merges independent chains into shared levels). A diamond has `span` 2 and `depth` 2; a
        # five-table chain has `span` 5 and `depth` 5 but a `critical_path` of five cheap costs.
        return max((height[1] for height in self._heights().values()), default=0)

    @property
    def severed(self) -> Option[PlanFault]:
        # a `dangling` parent `graphlib` silently absorbs as already-done, or a `cyclic` set it refused to
        # order — either makes the front order a lie, so the verdict carries WHICH cause (and its offending
        # keys) the root acts on: `cyclic` first (the harder structural break), then `dangling`, `Nothing`
        # for a sound plan. A `bool` collapsing both erases the cause the root needs to repair the right edge.
        if self.cyclic:
            return Some(PlanFault(cyclic=self.cyclic))
        return Some(PlanFault(dangling=self.dangling)) if self.dangling else Nothing

class ArtifactPipeline(Struct, frozen=True):
    nodes: Block[ArtifactWork]
    lane: LanePolicy = _DEFAULT_LANE
    cache_seed: Map[ContentKey, ArtifactReceipt] = Map.empty()

    async def plan(self, /) -> RuntimeRail[PipelinePlan]:
        # the planner is the third `core/receipt#RECEIPT` `contribute` consumer: a `planned`-phase fact rides
        # the one receipt stream the reuse-fabric elision and the `MeterProvider` read, so the planning stage
        # is observable WITHOUT a parallel plan-receipt rail — `Receipt.of("artifacts", ("planned", ...))`
        # mints the `fact` case the receipts owner's `PHASE_LEVEL` logs at `debug`, the `.map` firing it only
        # on a resolved plan so a `severed`/coverage verdict still rides the rail rather than a blind emit.
        railed = await async_boundary("pipeline.plan", self._plan)
        return railed.map(self._observed)

    def _observed(self, plan: PipelinePlan, /) -> PipelinePlan:
        # the planned-phase facts map is built inline here, not a single-call `_evidence` hop: the native
        # int coverage shape the receipts `enc_hook=repr`/`order="deterministic"` renderer serializes
        # unstringified, the `severed` verdict projected to its cause tag (or `"ok"`) so the line carries
        # WHICH coverage failure rather than a bare count.
        facts: dict[str, object] = {
            "cardinality": plan.cardinality, "live": plan.live, "depth": plan.depth, "width": plan.width,
            "contention": plan.contention, "critical_path": plan.critical_path, "span": plan.span,
            "elided": len(plan.elided), "roots": len(plan.roots), "leaves": len(plan.leaves),
            "severed": plan.severed.map(lambda fault: fault.tag).default_value("ok"),
        }
        Signals.emit(Receipt.of("artifacts", ("planned", "pipeline.plan", facts)), _PLAN_FACTS)
        return plan

    async def _plan(self, /) -> PipelinePlan:
        by_key: Map[ContentKey, ArtifactWork] = Map.of_seq((node.key, node) for node in self.nodes)
        requires: Map[ContentKey, tuple[ContentKey, ...]] = Map.of_seq((node.key, node.parents) for node in self.nodes)
        weights: Map[ContentKey, float] = Map.of_seq((node.key, node.cost) for node in self.nodes)
        keys = frozenset(node.key for node in self.nodes)
        dangling = frozenset(parent for node in self.nodes for parent in node.parents) - keys
        # the elision evidence: a `keyed` node whose content key already resolved into the threaded
        # session cache replays the upstream `Ok` in the runtime drain rather than re-rendering, so
        # the plan reports which keys will short-circuit before the lane probes a single one.
        elided = frozenset(node.key for node in self.nodes if node.admission.tag == "keyed") & frozenset(self.cache_seed.keys())
        order: TopologicalSorter[ContentKey] = TopologicalSorter()
        for node in self.nodes:
            order.add(node.key, *node.parents)
        try:
            order.prepare()
        except CycleError as cycle:
            # `CycleError.args[1]` is the cyclic node chain; surface it as evidence and return an
            # empty-front plan the `severed` verdict refuses, never a raise escaping the projection.
            return PipelinePlan(fronts=(), requires=requires, weights=weights, lane=self.lane, cache_seed=self.cache_seed, keys=keys, dangling=dangling, elided=elided, cyclic=frozenset(cycle.args[1]))
        # the stateful graphlib active driver (`is_active`/`get_ready`/`done`) is the one interop loop;
        # depth is in fronts not nodes (no async recursion), `fronts` rebound immutably per front, a
        # dangling `get_ready` key dropping through `by_key.try_find`. The plan ORDERS, the lane DRIVES.
        fronts: Block[Block[Admit[ArtifactReceipt]]] = Block.empty()
        while order.is_active():
            front = order.get_ready()
            units = Block.of_seq(front).choose(lambda key: by_key.try_find(key).map(lambda node: node.lowered()))
            fronts = fronts.append(Block.singleton(units))
            order.done(*front)
        return PipelinePlan(fronts=tuple(fronts), requires=requires, weights=weights, lane=self.lane, cache_seed=self.cache_seed, keys=keys, dangling=dangling, elided=elided)

    @staticmethod
    def of(works: ArtifactWork | Iterable[ArtifactWork], *, lane: LanePolicy = _DEFAULT_LANE, warm: Map[ContentKey, ArtifactReceipt] = Map.empty()) -> "ArtifactPipeline":
        nodes = Block.singleton(works) if isinstance(works, ArtifactWork) else Block.of_seq(works)
        return ArtifactPipeline(nodes=nodes, lane=lane, cache_seed=warm)
```
