# [LANDING]

The landing doctrine: how a run finding becomes durable law, at which surface, and the bar every landing clears. Every doctrine lander reads this page with the README `[02]-[ADMISSION]` table before its first edit; the reviewer configs derive their doctrine-surface guidance from it, so a diff landing doctrine below this bar is a reviewer finding.

## [01]-[BAR]

- A landing is one of three moves, exhausted in order: HARDEN an existing clause in place (sharpen, correct, densify), EXTEND an existing file with a new row or clause inside the owner, MINT a new file. Each move requires proof the prior move cannot carry the fact.
- Admission evidence is recurrence across independent units, or a structural singleton whose blast radius spans folders or branches; a unit-local lesson never lands.
- Landing nothing is a first-class verdict — a nomination refuted, already-owned, or gate-enforceable is rejected with its reason in the run report.
- A landed row states law in the timeless register: no run provenance, no freshness, no origin, no process narration; the run evidence stays in the lander's report.
- A rule a formatter, analyzer, or gate can enforce routes to that gate owner, never to reviewer prose or a doctrine page.

## [03]-[LAWS_PAGE_GRAMMAR]

- The default entry is one atomic table row. A law that cannot state itself inside a row's cell budget graduates to a `- [SLUG]:` entry in the page's `[DEEP_ROWS]` section — two to four sentences, marker-led, never a heading-per-item card and never a prose essay.
- One page holds one law family; a new page requires a family no page charters and lands its README pages-table row in the same change.
- Rows are re-proven, never archived: heavy runs verify topology rows against the live tree and cull dead ones; a scar moves out the moment a gate or doctrine page absorbs its law; a patterns row that stops binding two branches demotes to the surviving branch's stack page.
- A law row names real repo surfaces in exact code-span spellings; an illustrative shape inside a row uses the neutral placeholder alphabet the standards own.

## [04]-[POISON_GUARD]

Every landing surface is context for future writers, so the anchoring laws bind the lander: no member rosters that freeze a census, no forbidden-form litanies, no counts as ceilings — the extension rule stands in the anchor's place; a rewrite never inherits the source frame. The lander runs the prose gate on every touched page and repairs to zero FAILs before returning.
