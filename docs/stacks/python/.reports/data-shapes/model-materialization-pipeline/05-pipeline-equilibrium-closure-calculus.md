# Closure Row Shape

- Every pipeline equilibrium obligation declares one frozen row: `closure_id`, `invariant_kind`, `primary_table`, `primary_row_id`, `foreign_table_ids`, `foreign_row_ids`, `proof_layers`, `collapse_row_id`, `blocking_merge`.

- `invariant_kind` draws from closed vocabulary ‚Äî `{handoff_surface_parity, attribution_transition_parity, smoke_signal_parity, consumer_evolution_parity, singleton_identity, promotion_atomicity, plane_stage_consistency}` ‚Äî not free-form checklist items.

- `primary_row_id` and `foreign_row_ids` are closed foreign-key slugs ‚Äî erased references or implicit name equality between stage tags and `transition_id` fail closure admission.
- `proof_layers` stacks orthogonal gates from ingress-to-materialization doctrine ‚Äî static import architecture, compiled oracle snapshot, lattice contract parametrization, metamorphic round-trip, runtime smoke injection ‚Äî failures at earliest layer block later layers.
- `collapse_row_id` names repair morphism when closure drifts ‚Äî undocumented collapse is merge blocker when `blocking_merge=true`.
- `blocking_merge` defaults `true` for production-admitted seams ‚Äî exemptions require composition-root registry entry citing `closure_id`.
