# [PY_ARTIFACTS_RULINGS]

Per-folder decision registry for the artifact engine — the settled rulings agents re-litigate for lack of a home. Each row is one decision with the one why guarding its re-litigation: a package admission or rejection, an owner-choice discriminant, a collapse refusal naming the lost guarantee, a structure retirement, or a working law. Append-mostly; a row earns its seat while its why stays homeless, dies only when fact and why both live at one durable surface, and a decision spanning folders homes at the branch tier. `(none)` marks a section awaiting its first ruling.

## [01]-[PACKAGES]

- Host-coupled rendering admits per-FORMAT, never per-ENGINE — a format hole on a host-free engine earns one gated opt-in host path whose wedge-kill the driver owns, because the format gate patches one hole on an already content-keyable engine while a per-engine host admission couples every artifact's existence and identity to host state.
- Graph-layout engines are in-process typed libraries whose results read back into the stable node index — graphviz/pygraphviz stays rejected: a DOT subprocess round-trip is stringly ingress and egress that cannot seat ports on `Port.at`, and the admitted engines already own orthogonal routing, port nesting, and crossing minimization.

## [02]-[SHAPE]

- Absent, empty, and zero stay three distinct facts — absence rides `Option`/`UnsetType`, never a truthiness fold: a `value or default` collapse fabricates PRESENT where PDF/UA auditing must read ABSENT, erases a real `0` form value at lens recovery, and conflates reserved with unknown at classification.
- Capability and route selection derive from probes and membership tables, never a caller-visible flag on a payload — a route flag enters content identity and lets callers mint build-dependent keys for one identical artifact.
- Substitute resolution walks a fidelity ladder — sibling native filter first, in-process composite second, numpy kernel last, the closest arm preserving the admitted payload: flattening to numpy forfeits libavfilter's colorspace-correct SIMD arms, and forcing native-always loses RAQM complex-script shaping.
- Admission modality derives from what the emission IS, never producer preference — an artifact signature over content-determined input admits `keyed` (artifact identity is idempotent and the cached signature is the correct bytes, since re-signing identical content mints N signatures for one artifact); an act or secret-consuming emission admits `bare`, evidencing an occurrence that never elides; transient admits `retried` with its `RetryClass`.
- Any new `Admission` case earns existence only with a behaviorally distinct lowering — a member without one is a phantom the vocabulary refuses.
- Producer admission speaks one idiom per rail — a head whose refusals fold through the folder `_FAULTS` boundary admits via `@beartype(conf=_INGRESS)` with `violation_type=ValueError`, and a head riding the runtime classify rail applies the imported `FAULT_CONF`, never a locally re-minted conf; a third per-producer dialect forks the refusal taxonomy callers must catch.
- XML rails split on trust origin — an untrusted packet admits through the one hardened `lxml` parse fold, and a self-generated fragment builds with the in-page tree builder (`lxml` where the build queries by XPath or class token, stdlib `ElementTree` for plain assembly); hardening a trusted fragment adds no guarantee, and a second hardened stack forks the admission every auditor must re-verify.
- Every collection whose iteration order reaches a content-key preimage or an output byte stream is deterministically ordered — key-sorted `Map`, `sorted`, or `dict.fromkeys`; a bare `set` or hash-order iteration forks the key per hash seed and breaks the elision cache the whole keyed-plan regime depends on.
- Receipt facts MEASURE the delivered product — a re-probe of the muxed bytes, or the delivery walk when the product is unprobeable; request policy enters only as labeled policy facts, because an echoed request knob attests the input, never the artifact.
- Batch admission accumulates every casualty through an associative fault monoid — one refusal reports every failed gate; first-failure abort hides the sibling casualties, and a `Result.to_option` erasure discards the fault evidence the refusal owes its caller.
- Producer heads split on input origin — a head normalizing an already-typed op union raises through its folder-configured ingress conf, and a head admitting untrusted external payload returns `Result[Self, <Fault>]`; unifying the two families flattens either the caller's typed refusal or the payload fault evidence, and the ingress conf mints once per folder — a second same-valued conf under a new name is the rejected fork.
- Folds offloading caller-programmable input rail the builder's `ValueError` as `invalid` — an admission raise reachable from a public case payload is caller data, never a programming defect, so the composing fold carries the `ValueError` arm its sibling folds hold; surfacing it as an opaque worker fault strips the refusal evidence the caller owes.
- Producer worker offload rides the producer's own `lane: LanePolicy` instance field through `self.lane.offload` — a class-qualified `LanePolicy.offload` carries no bound instance, and a folder-minted `CapacityLimiter` oversubscribes the host against the shared runtime process band.

## [03]-[COLLAPSE]

- `drawsvg` and `ezdxf` stay two lowerings per mark — DXF is the editable associative CAD contract (live dimension variables, one-authored-N-placed blocks, CAD-editable layer tables) and SVG the font-independent, OCG-bindable print vector (outlined text, named groups); neither contract nests in the other, and rasterizing to one form destroys both.
- Both subtitle burn arms stay separate — the filtergraph burn is the chain-position banner over flat timed events mid-transcode, the subtitle burn the document-faithful render (styled runs, numpad anchoring, margins) over ndarray frames pre-encode; merging drags document semantics into the chain builder or forfeits the streaming one-pass burn.
- Package plane keeps codec, archive, and delta as three producers — their recovery models are structurally incompatible (self-delimiting frame walk, central-directory member recovery, parent-keyed patch application), and a single polymorphic packer also collapses per-executor enforcement: the terminal kill-arm on untrusted decode, the witnessed delta round-trip, and the archive integrity drain.
